# 🎬 B站数据爬虫 (ReptileBilibili)

一个基于 Python + Requests 的哔哩哔哩数据爬虫工具，支持扫码登录，可获取用户的观看历史、关注列表、收藏视频、番剧订阅等多种数据。

## ✨ 功能特性

- 🔐 **扫码登录** - 支持终端二维码扫码登录，自动保存登录状态
- 👤 **用户信息** - 获取用户基本信息、粉丝数、关注数等
- 📺 **观看历史** - 获取过去一周的观看历史记录
- 👥 **关注列表** - 获取所有关注的UP主列表
- 🎥 **UP主视频** - 获取关注UP主的最新视频和热门视频
- ⭐ **收藏视频** - 获取所有收藏夹中的视频
- 📖 **番剧订阅** - 获取订阅的番剧/影视列表
- 👍 **点赞视频** - 获取最近点赞的视频（100个）
- 🪙 **投币视频** - 获取最近投币的视频（100个）

## 🚀 快速开始

### 环境要求

- Python >= 3.13

### 安装依赖

```bash
# 使用 pip
pip install requests qrcode

# 或使用 uv (推荐)
uv sync
```

### 运行程序

```bash
python main.py
```

### 使用流程

1. 运行程序后，终端会显示登录二维码
2. 使用手机哔哩哔哩 APP 扫描二维码登录
3. 登录成功后进入功能菜单
4. 选择需要的功能进行数据爬取
5. 爬取的数据会保存在 `data/` 目录下

## 📊 数据输出

爬取的数据将以 CSV 格式保存在 `data/` 目录下，便于后续分析和处理。

| 功能 | 输出文件 |
|------|----------|
| 观看历史 | `history_videos.csv` |
| 关注列表 | `follow_list.csv` |
| 收藏视频 | `favorite_videos.csv` |
| 番剧订阅 | `bangumi_list.csv` |
| 点赞视频 | `liked_videos.csv` |
| 投币视频 | `coined_videos.csv` |
| 用户信息 | `user_info.json` |

## ⚠️ 注意事项

1. **登录凭证**：登录成功后，Cookies 会保存在 `cookies.json` 中，下次运行无需重新登录
2. **请求频率**：程序内置了请求间隔，请勿修改以避免触发风控
3. **仅供学习**：本项目仅供学习交流使用，请勿用于商业用途
4. **合理使用**：请遵守哔哩哔哩的用户协议，合理使用本工具
