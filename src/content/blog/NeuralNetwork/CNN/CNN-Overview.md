## 1. 全连接网络处理图像的问题

假设我们有一张 $28 \times 28$ 的灰度手写数字图片。如果使用传统的全连接神经网络（MLP），第一步就是把这个 $28 \times 28 = 784$ 的二维矩阵"拉平"成一个 784 维的向量。

这样做会带来两个严重问题：

**问题一：参数量爆炸**

第一层隐藏层如果有 128 个神经元，那么光是第一层就有 $784 \times 128 + 128 = 100,480$ 个参数。如果图像尺寸增加到 $224 \times 224 \times 3$（一张彩色照片），输入维度就是 150,528，第一层 128 个神经元就需要 $150,528 \times 128 \approx 1900$ 万个参数。这对于训练数据和计算资源都是巨大的负担。

**问题二：丢失空间信息**

把二维图像拉平成一维向量，等于完全丢弃了像素之间的空间位置关系。在图像中，相邻像素通常高度相关——比如"7"这个数字，就是由特定位置上的笔迹构成的。拉平操作让网络无法利用这种空间结构。

## 2. CNN 的两个关键设计

CNN 通过两个关键设计解决上述问题：

### 局部连接（局部感受野）

每个神经元只连接输入的一个小区域（比如 $3 \times 3$ 或 $5 \times 5$），而不是全部像素。这样参数数量大幅减少，且自然保留了空间局部性。

### 权重共享

同一个卷积核（滤波器）在整个图像上滑动，用同一组参数检测不同位置的相同特征。如果用一个 $3 \times 3$ 的卷积核检测"边缘"，无论边缘出现在图像的左上角还是右下角，都能被检测到。

## 3. 从"特征工程"到"特征学习"

传统图像识别流程是：

```
原始图像 → 人工设计特征提取器（SIFT, HOG等）→ 分类器（SVM等）→ 结果
```

CNN 的做法是：

```
原始图像 → 网络自动学习特征 → 分类 → 结果
```

CNN 不需要人工设计特征，而是让网络自己从数据中学习"什么样的特征对分类有用"。这种端到端的学习方式，是深度学习革命的核心。

## 4. 层次化特征学习

这是 CNN 最深刻的思想：**特征是有层次结构的**。

- **浅层（靠近输入）** 学到的是低级特征：边缘、角点、颜色、纹理方向
- **中层** 学到的是中级特征：由边缘组合而成的形状、纹理图案
- **深层（靠近输出）** 学到的是高级特征：由形状组合而成的物体部件、整体轮廓

以手写数字 "8" 为例：
- 浅层检测各个方向的边缘段
- 中层检测弧线、圆圈的一部分
- 深层检测"两个圆圈叠在一起"这个整体结构

## 5. CNN 的五大基本组件

任何一个 CNN 都由以下组件堆叠而成：

| 组件                                                             | 作用      |  是否改变空间尺寸   |
| -------------------------------------------------------------- | ------- | :---------: |
| 卷积层（Conv）  | 提取特征    | 取决于 padding |
| 激活函数（Activation） | 引入非线性   |      否      |
| 池化层（Pooling）   | 下采样，降维  |    是（缩小）    |
| 批量归一化（BatchNorm）     | 稳定训练    |      否      |
| 全连接层（FC）                              | 综合特征做分类 |  N/A（接收向量）  |
一个典型的 CNN 架构模式是：

$$
[\text{Conv} \to \text{Activation} \to \text{Pool}] \times N \to \text{Flatten} \to [\text{FC} \to \text{Activation}] \times M \to \text{Output}
$$

## 6. CNN 的发展脉络

| 年份 | 架构 | 核心创新 |
| --- | --- | --- |
| 1998 | LeNet-5 | 奠定 CNN 基础模式（Conv→Pool→FC） |
| 2012 | AlexNet | ReLU 激活、Dropout、GPU 训练 |
| 2014 | VGG | 全部使用 3×3 小卷积核，网络更深 |
| 2014 | NiN | MLP 卷积、用 GAP 替代全连接层 |
| 2014 | GoogLeNet | Inception 多分支结构，22 层仅 7M 参数 |
| 2015 | ResNet | 残差学习解决退化问题，可训练 152+ 层 |
