## 第2题-统计二叉树中"平衡路径"的数量（P4821）| 中等 | 1000ms

### 题目描述

定义二叉树的平衡路径需同时满足以下 3 个条件：

1. 路径从任意节点出发，仅能向下延伸（只能向左/右子节点，不可向上回溯）。
2. 路径上所有节点的和相加为 0。
3. 路径长度（包含的节点个数）至少为 2。

请实现一个 Python 函数，输入二叉树的根节点（按层序遍历规则构建），返回该树中所有"平衡路径"的总数。

**注意**：
- 建树规则：层序遍历列表按「从上到下、从左到右」的顺序构建二叉树，None 表示对应位置无节点。
- 路径延伸：从起点出发，仅沿左子节点 OR 右子节点单向向下（单链，不可分叉）。
- 统计方式：每个符合条件的单链路径独立计数（即使路径有重叠）。

### 输入格式

二叉树的层序遍历列表（元素为整数或 None，None 表示空节点）。

### 输出格式

整数，表示平衡路径的总数。

### 样例 1

**输入**
```
[10, -5, -5, 2, -2, 3, -3]
```

**输出**
```
0
```

### 样例 2

**输入**
```
[0, 0, None]
```

**输出**
```
1
```

### 样例 3

**输入**
```
[1, -1, 2, -2, None, 3, -3]
```

**输出**
```
2
```

## 解题思路
从任意节点出发，DFS遍历所有向下的单链路径（每步只能选择左子节点或右子节点），跟踪路径和与长度。当路径长度≥2且和为0时计数。对树中每个可能的起始节点执行此DFS，累计所有满足条件的路径总数。

## 参考代码
```python
import sys

def buildTree(arr):
    """从层序遍历列表构建二叉树，返回节点列表"""
    if not arr or arr[0] is None:
        return None
    nodes = []
    for val in arr:
        if val is None:
            nodes.append(None)
        else:
            nodes.append([val, None, None])  # [value, leftIdx, rightIdx]
    rootIdx = 0
    queue = [rootIdx]
    idx = 1
    while queue and idx < len(arr):
        currIdx = queue.pop(0)
        if nodes[currIdx] is None:
            continue
        # 左子节点
        if idx < len(arr):
            nodes[currIdx][1] = idx
            queue.append(idx)
            idx += 1
        # 右子节点
        if idx < len(arr):
            nodes[currIdx][2] = idx
            queue.append(idx)
            idx += 1
    return nodes

def countBalancedPaths(nodes, startIdx):
    """从startIdx出发，统计所有满足条件的向下路径数"""
    ans = 0

    def dfs(nodeIdx, runningSum, length):
        nonlocal ans
        node = nodes[nodeIdx]
        if node is None:
            return
        newSum = runningSum + node[0]
        newLen = length + 1
        if newLen >= 2 and newSum == 0:
            ans += 1
        # 向左延伸
        if node[1] is not None:
            dfs(node[1], newSum, newLen)
        # 向右延伸
        if node[2] is not None:
            dfs(node[2], newSum, newLen)

    dfs(startIdx, 0, 0)
    return ans

def solve():
    data = sys.stdin.read().strip()
    arrStr = data.strip()
    # 去除方括号
    if arrStr.startswith('['):
        arrStr = arrStr[1:]
    if arrStr.endswith(']'):
        arrStr = arrStr[:-1]
    parts = arrStr.split(',')
    arr = []
    for p in parts:
        p = p.strip()
        if p == 'None' or p == 'null' or p == '':
            arr.append(None)
        else:
            arr.append(int(p))

    nodes = buildTree(arr)
    if nodes is None:
        print(0)
        return

    total = 0
    for i in range(len(nodes)):
        if nodes[i] is not None:
            total += countBalancedPaths(nodes, i)
    print(total)

if __name__ == '__main__':
    solve()
```

## 第3题-网络异常流量传播链路溯源（P4822）| 困难 | 1000ms

### 题目描述

在网络监控中，异常流量的流动通常具有局部聚集性。监控系统需要识别出高负载的基站（关键节点），并判断流量在这些节点之间定向传播链的最长路径。

#### 网络监控规则

**直接关联**：对于基站 A 和 B，若其曼哈顿距离 |xA - xB| + |yA - yB| ≤ ε_dist，则判定两者具有直接关联。

**关键节点判定**：计算一个基站及其所有具有"直接关联"属性的基站（含自身）的流量负载 w 之和。若该总和 ≥ W_threshold，则该基站被判定为关键节点。

#### 流量传播链路

流量只能在关键节点之间定向流动，规则如下：

**链路条件**：若两个关键节点具有"直接关联"关系，且发生时间戳 t 不同，则流量从时间较早的基站流向时间较晚的基站。

**注意**：若两个关联的关键节点发生时间完全相同，则它们之间无法建立有效的传播链路。

**传播链条**：传播链条是由一系列关键节点通过有向链路首尾相连构成的路径。

**衡量指标**：链条的规模为该路径上所有节点服务的用户数 Users 之和。

**任务**：计算全网中可能形成的所有传播链条中，能够覆盖的最大用户总数。

#### 处理流程

1. **节点识别**：基于空间距离和流量负载阈值，从所有基站中筛选出满足要求的关键节点。
2. **构建拓扑**：在关键节点间，基于空间关联和时间先后顺序（时间早 → 时间晚）建立定向传播关系。
3. **路径计算**：在构成的有向传播网络中，寻找一条或多条连续路径，使得累计服务的用户数 Users 达到最大。

### 输入格式

第 1 行：包含 3 个整数 N（基站总数，1 ≤ N ≤ 200）、ε_dist（空间阈值）和 W_threshold（负载阈值）。

第 2 行到 N+1 行，每行包含 5 个整数：x, y, t, w, Users。所有坐标、时间戳、负载和用户数的取值范围均为 [0, 10^9]。

### 输出格式

输出一个整数，代表最大用户数。若全网无法形成任何链路或关键节点，输出 0。

### 样例 1

**输入**
```
3 1 500
0 0 10 100 50
1 0 20 100 50
0 1 30 100 50
```

**输出**
```
0
```

**说明**：三个基站虽然互为邻居，但每个基站能查到的邻域最大负载和仅为 100 × 3 = 300。判定门槛 W_threshold = 500，因 300 < 500，图中没有任何基站能成为关键节点。由于传播链路只能在关键节点间建立，无关键节点即无法形成链条，输出 0。

### 样例 2

**输入**
```
4 1 150
0 0 10 100 10
1 0 20 100 10
5 5 10 200 100
5 6 30 200 100
```

**输出**
```
200
```

**说明**：基站 0 和 1 曼哈顿距离为 1，互为邻居，各自负载均为 100 + 100 = 200 > 150，均为关键节点。基站 2 和 3 曼哈顿距离为 1，负载 200 + 200 = 400 > 150，均为关键节点。时间上 0→1（10→20）可形成链路，用户数为 10 + 10 = 20；2→3（10→30）用户数为 100 + 100 = 200。最大用户数为 200。

## 解题思路
先根据曼哈顿距离判定每个基站的邻域负载总和，筛选出关键节点。然后在关键节点间按时间先后（t_i < t_j）建立有向边，形成DAG。对DAG做拓扑排序后用DP求最长路径（以节点用户数为权重），dp[v] = users[v] + max(dp[前驱])。最终答案为所有存在前驱的节点中dp的最大值，若无有效链路则输出0。

## 参考代码
```python
import sys

def solve():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    it = iter(data)
    N = int(next(it))
    eps = int(next(it))
    W_threshold = int(next(it))

    stations = []  # (x, y, t, w, users)
    for _ in range(N):
        x = int(next(it)); y = int(next(it)); t = int(next(it))
        w = int(next(it)); users = int(next(it))
        stations.append((x, y, t, w, users))

    # 步骤1：识别关键节点
    keyNodes = []
    for i in range(N):
        xi, yi, ti, wi, ui = stations[i]
        totalW = 0
        for j in range(N):
            xj, yj, tj, wj, uj = stations[j]
            if abs(xi - xj) + abs(yi - yj) <= eps:
                totalW += wj
        if totalW >= W_threshold:
            keyNodes.append(i)

    if len(keyNodes) < 2:
        print(0)
        return

    # 步骤2：构建有向图
    keySet = set(keyNodes)
    adj = [[] for _ in range(N)]
    origIndegree = [0] * N

    for i in keyNodes:
        for j in keyNodes:
            if i == j:
                continue
            dist = abs(stations[i][0] - stations[j][0]) + abs(stations[i][1] - stations[j][1])
            if dist <= eps and stations[i][2] < stations[j][2]:
                adj[i].append(j)
                origIndegree[j] += 1

    # 检查是否至少存在一条边
    hasEdge = any(len(adj[i]) > 0 for i in range(N))
    if not hasEdge:
        print(0)
        return

    # 步骤3：拓扑排序
    indegree = origIndegree[:]
    q = [i for i in keyNodes if indegree[i] == 0]
    topo = []
    while q:
        u = q.pop(0)
        topo.append(u)
        for v in adj[u]:
            indegree[v] -= 1
            if indegree[v] == 0:
                q.append(v)

    # 步骤4：DP求最大用户数路径
    dp = [0] * N
    for i in keyNodes:
        dp[i] = stations[i][4]

    for u in topo:
        for v in adj[u]:
            if dp[u] + stations[v][4] > dp[v]:
                dp[v] = dp[u] + stations[v][4]

    # 答案：存在前驱的节点中dp最大值
    ans = 0
    for i in keyNodes:
        if origIndegree[i] > 0:
            ans = max(ans, dp[i])

    print(ans)

if __name__ == '__main__':
    solve()
```
