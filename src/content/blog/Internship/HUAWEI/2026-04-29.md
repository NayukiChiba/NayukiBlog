## 第1题-选择题（P4868）共20道，1-15单选，16-20多选

### 1. 非负矩阵分解（NMF）的适用场景是？
- A. 数据含负值的场景
- B. 矩阵可逆的场景
- C. 数据非负且需解释性的场景（如文本聚类）
- D. 高维数据降维场景

:::fold[答案]

正确答案：C

解析：NMF要求分解后的矩阵元素均为非负，天然适合非负数据（如词频、像素值），非负约束使得分解结果具有部分-整体可解释性，常用于文本主题建模和图像部件分解。
:::

### 2. 矩阵的条件数$\text{cond}(A)$的定义为
- A. $\lVert A^{-1} \rVert / \lVert A \rVert$
- B. $\lVert A \rVert / \lVert A^{-1} \rVert$
- C. $\lVert A \rVert + \lVert A^{-1} \rVert$
- D. $\lVert A \rVert \cdot \lVert A^{-1} \rVert$

:::fold[答案]

正确答案：D

解析：矩阵条件数定义为 $\text{cond}(A) = \lVert A \rVert \cdot \lVert A^{-1} \rVert$，衡量线性方程组对扰动的敏感程度。条件数越大，矩阵越病态。
:::

### 3. MobileNet核心创新点在于将标准卷积分解为
- A. 空间卷积和时间卷积
- B. 深度卷积（Depthwise）和逐点卷积（Pointwise）
- C. $1 \times 1$卷积和$3 \times 3$卷积的并行计算
- D. 全连接层和池化层的组合

:::fold[答案]

正确答案：B

解析：MobileNet将标准卷积分解为Depthwise卷积（每个通道独立进行空间卷积）和Pointwise卷积（$1 \times 1$跨通道融合），大幅减少参数量和计算量。
:::

### 4. 关于线性判别分析（LDA），下列说法正确的是
- A. LDA是无监督降维方法
- B. LDA的目标是最大化类间散度与类内散度的比值
- C. LDA降维后的维度可以任意指定，最多等于特征数
- D. LDA适用于任何类型的数据分布

:::fold[答案]

正确答案：B

解析：LDA是有监督降维方法，目标是最大化类间散度矩阵 $S_B$ 与类内散度矩阵 $S_W$ 的比值。降维后维度最多为 $C-1$（$C$为类别数），且假设各类服从同方差高斯分布。
:::

:::

:::

### 12. AdamW中"W"代表什么改进？

### 14. 数值微分：前向差分、中心差分和Richardson外推法的误差阶数分别是？

- B. 前向$O(h^2)$，中心$O(h)$，Richardson $O(h)$

## 第2题-基于LSTM进行室内温度预测（P4869）| 中等 | 1000ms

### 题目背景

在现代智能家居系统中，准确预测室内温度变化对于实现舒适的居住环境和节能控制至关重要。室内温度受多种因素影响，包括室外温度、空调设定温度、门窗开关状态、室内人员数量等，这些因素之间存在复杂的非线性关系。

假设你正在开发一个智能温控系统，该系统通过传感器采集过去一段时间内的环境数据，使用 LSTM 模型预测未来时刻的室内温度。具体来说，系统每 5 分钟采集一次数据，包括以下 5 个特征：

1. 室内温度
2. 室外温度
3. 空调功率
4. 门窗开合状态
5. 室内人员数量

### 任务

请你基于过去 T 个时间步的这 5 个特征，使用 LSTM 模型学习温度变化的动态模式，输出每个时间步的隐藏状态和最终细胞状态。

### LSTM 核心公式

给定输入 $x_t$、上一时刻隐藏状态 $h_{t-1}$、上一时刻细胞状态 $c_{t-1}$，$\sigma()$ 为 sigmoid 激活函数：

- 遗忘门：$f_t = \sigma(W_f \cdot [h_{t-1}, x_t] + b_f)$
- 输入门：$i_t = \sigma(W_i \cdot [h_{t-1}, x_t] + b_i)$
- 候选细胞状态：$\tilde{c}_t = \tanh(W_c \cdot [h_{t-1}, x_t] + b_c)$
- 细胞状态更新：$c_t = f_t \odot c_{t-1} + i_t \odot \tilde{c}_t$
- 输出门：$o_t = \sigma(W_o \cdot [h_{t-1}, x_t] + b_o)$
- 隐状态更新：$h_t = o_t \odot \tanh(c_t)$
- 输出层：$\hat{y}_t = W_y \cdot h_t + b_y$

本题中，上一时刻隐藏状态 $h_{t-1}$、上一时刻细胞状态 $c_{t-1}$ 均初始化为 0。

### 输入描述

第一行：`T B D H` 共 4 个正整数，其中 T 为时间步数，B 为批次大小（独立房间数），D 为输入特征维度（固定为 5），H 为隐藏层维度。取值范围：$0 < T \le 100,\; 0 < B \le 100,\; D = 5,\; 0 < H \le 10$。

接下来 T 行：每行包含 $B \times D$ 个浮点数，表示输入序列，按时间步顺序，每个时间步的数据按批次顺序排列（即先第 1 个房间的 D 个特征，再第 2 个房间的 D 个特征，依此类推）。

最后四行：分别对应遗忘门、输入门、输出门和候选细胞状态的参数，每行包含：

- 输入权重（$D \times H$）、隐藏权重（$H \times H$）、偏置（$H$）的元素
- 所有参数按行优先顺序展平

注：所有输入每个数之间用一个空格分隔，每行末尾无多余空格。

### 输出描述

- 第一行：所有时间步的隐藏状态（$T \times B \times H$），按时间步顺序展平输出
- 第二行：最终细胞状态（$B \times H$），按批次顺序展平输出
- 所有浮点数保留 4 位小数（四舍五入），每个数之间用一个空格分隔，每行末尾无多余空格

### 样例 1

**输入**

```
1 1 5 2
2.5 5.0 3.2 5.0 1
0.1 0.1 0.3 3.4 0.5 0.4 0.7 0.3 0.2 1.0 0.7 0.5 0.8 1.0 1.1 1.4 0.1 0.7 0.1 1.4
0.5 0.6 0.3 0.2 0.3 1.0 0.8 0.7 0.7 1.1 1.2 1.2 0.1 0.8 3.3 0.4 0.1 0.5 0.4 0.8
0.9 1.0 0.6 0.9 0.5 1.2 1.1 1.5 0.1 0.1 0.3 0.4 0.5 0.6 0.6 0.8 0.2 1.0 0.7 0.1
0.9 1.3 1.6 1.2
```

**输出**

```
0.7615 0.7616
0.9997 1.0000
```

**说明**

第一行：$T=1,\;B=1,\;D=5,\;H=2$：序列长度 1（1 个采样点），批次大小为 1（1 个独立房间），每个房间 5 维特征，隐藏层维度 2。

第二行：$1 \times 5$ 大小的输入序列，房间1的特征：`[2.5 5.0 3.2 5.0 1]`

最后 4 行：各门参数，前 $5 \times 2$ 个为输入权重，其后 $2 \times 2$ 个为隐藏权重，最后 2 个为偏置项。

输出第一行：$1 \times 1 \times 2$ 个数据，序列 1 的隐藏状态 `[0.7615 0.7616]`
输出第二行：最终细胞状态，输出 $1 \times 2$ 个数据 `[0.9997 1.0000]`

### 样例 2

**输入**

```
2 2 5 1
21.0 25.0 22.0 25.5 1 21.0 25.0 22.0 25.5 2
22.0 26.0 20.5 24.0 2 22.0 26.0 20.5 24.0 1
0.1 0.15 0.2 0.1 0.15 0.3 0.3
0.4 0.45 0.5 0.4 0.45 0.5 0.6
0.7 0.6 0.8 0.7 0.6 0.8 0.9
1.0 0.9 1.1 1.0 0.9 1.1 1.2
```

**输出**

```
0.7616 0.7616 0.9640 0.9640
2.0000 2.0000
```

**说明**

$T=2,\;B=2,\;D=5,\;H=1$：序列长度为 2（2 个采样点），批次大小为 2（2 个独立房间），每个房间 5 维特征，隐藏层维度为 1。

接下来两行：分别为两个时间步的输入序列，每个时间步包含 2 个房间的 5 维特征。

最后 4 行：各门参数，包含遗忘门、输入门、输出门和候选细胞状态的权重与偏置。

输出第一行：按时间步顺序展平输出所有时间步的隐藏状态 $2 \times 2 \times 1$，结果为 `0.7616 0.7616 0.9640 0.9640`。
输出第二行：按批次顺序展平输出最终细胞状态 $2 \times 1$，结果为 `2.0000 2.0000`。

## 解题思路
按LSTM公式从零实现前向传播：对每个时间步，拼接当前输入 $x_t$ 和上一时刻隐藏状态 $h_{t-1}$，分别计算遗忘门、输入门、候选细胞状态和输出门（使用给定的权重矩阵和偏置），然后更新细胞状态 $c_t$ 和隐藏状态 $h_t$。输入权重矩阵为 $D \times H$（行优先），隐藏权重为 $H \times H$（行优先）。输出所有时间步隐藏状态和最终细胞状态，保留4位小数。

## 参考代码
```python
import sys
import math

def sigmoid(x):
    """sigmoid激活函数，带溢出保护"""
    if x > 20:
        return 1.0
    if x < -20:
        return 0.0
    return 1.0 / (1.0 + math.exp(-x))

def tanhClipped(x):
    """tanh激活函数，带溢出保护"""
    if x > 20:
        return 1.0
    if x < -20:
        return -1.0
    return math.tanh(x)

def solve():
    data = sys.stdin.read().strip().split()
    it = iter(data)
    T = int(next(it))
    B = int(next(it))
    D = int(next(it))
    H = int(next(it))

    # 读取输入序列：T行，每行B*D个浮点数
    X = []  # X[t][b][d]
    for t in range(T):
        step = []
        for b in range(B):
            feat = [float(next(it)) for _ in range(D)]
            step.append(feat)
        X.append(step)

    # 读取4个门的参数（遗忘门、输入门、输出门、候选细胞状态）
    # 每个门：D*H个输入权重 + H*H个隐藏权重 + H个偏置
    gateParams = []
    for _ in range(4):
        Wi = [float(next(it)) for _ in range(D * H)]   # 输入权重 D×H 行优先
        Wh = [float(next(it)) for _ in range(H * H)]   # 隐藏权重 H×H 行优先
        b = [float(next(it)) for _ in range(H)]        # 偏置
        gateParams.append((Wi, Wh, b))

    def computeGate(gateIdx, xCurr, hPrev):
        """计算单个门的输出（未激活）"""
        Wi, Wh, bias = gateParams[gateIdx]
        result = [float(bias[hh]) for hh in range(H)]
        # 输入贡献：Wi[d * H + h] 是 x[d] 到 gate[h] 的权重
        for d in range(D):
            base = d * H
            for hh in range(H):
                result[hh] += Wi[base + hh] * xCurr[d]
        # 隐藏贡献：Wh[hh * H + h] 是 hPrev[hh] 到 gate[h] 的权重
        for prevH in range(H):
            base = prevH * H
            for hh in range(H):
                result[hh] += Wh[base + hh] * hPrev[prevH]
        return result

    # 初始化隐藏状态和细胞状态为0
    h = [[0.0] * H for _ in range(B)]  # B × H
    c = [[0.0] * H for _ in range(B)]  # B × H

    allH = []  # 每个时间步的展平隐藏状态

    for t in range(T):
        xt = X[t]  # B × D
        newH = [[0.0] * H for _ in range(B)]
        newC = [[0.0] * H for _ in range(B)]

        for b in range(B):
            xCurr = xt[b]
            hPrev = h[b]

            # 计算四个门（未激活）
            ftRaw = computeGate(0, xCurr, hPrev)  # 遗忘门
            itRaw = computeGate(1, xCurr, hPrev)  # 输入门
            otRaw = computeGate(2, xCurr, hPrev)  # 输出门
            cctRaw = computeGate(3, xCurr, hPrev) # 候选细胞状态

            # 激活
            ft = [sigmoid(v) for v in ftRaw]
            it = [sigmoid(v) for v in itRaw]
            ot = [sigmoid(v) for v in otRaw]
            cct = [tanhClipped(v) for v in cctRaw]

            # 细胞状态更新：c_t = f_t * c_{t-1} + i_t * cct_t
            for hh in range(H):
                newC[b][hh] = ft[hh] * c[b][hh] + it[hh] * cct[hh]

            # 隐藏状态更新：h_t = o_t * tanh(c_t)
            for hh in range(H):
                newH[b][hh] = ot[hh] * tanhClipped(newC[b][hh])

        # 记录本时间步隐藏状态
        stepFlat = []
        for b in range(B):
            stepFlat.extend(newH[b])
        allH.append(stepFlat)

        h = newH
        c = newC

    # 输出第一行：所有时间步隐藏状态 T×B×H
    out1 = []
    for t in range(T):
        out1.extend(allH[t])
    print(' '.join(f'{x:.4f}' for x in out1))

    # 输出第二行：最终细胞状态 B×H
    out2 = []
    for b in range(B):
        out2.extend(c[b])
    print(' '.join(f'{x:.4f}' for x in out2))

if __name__ == '__main__':
    solve()
```

## 第3题-大模型预训练语料清洗（P4870）| 中等 | 1000ms

### 题目背景

为了防止大语言模型（LLM）在生成时出现"复读机"现象，并确保训练语料的语义纯净，你需要模拟一个数据清洗引擎。给定 N 篇待清洗文档，你需要按顺序执行以下五大清洗规则，只有全部通过的文档才会被保留并进入最终的训练集。

### 清洗规则

**规则 1：空格规范化**

首先对文档中所有连续的空白字符（空格、制表符等）合并为一个空格，并去除首尾空格。规范化后的字符串总长度必须在 $[L, R]$ 的闭区间内，如果不满足，直接丢弃该文档。注意：如果文档被保留，最终输出的必须是这个"规范化后"的字符串。

**规则 2：语义单词提取**

将规范化后的文档按非字母数字字符（即除了 a-z、A-Z、0-9 以外的所有字符）进行分割，提取出所有的有效"单词"，并全部转化为小写。例如：`"He is AI Spammer."` 提取出的单词序列为 `["he", "is", "ai", "spammer"]`。

**规则 3：独立黑名单拦截**

给定 K 个黑名单词汇。如果在第 2 步提取出的单词序列中，精确匹配到了任何一个黑名单词汇（不区分大小写），则丢弃该文档。注意：必须是独立单词匹配，不能是子串匹配。例如黑名单中有 `spam`，但文档中的 `spammer` 可以是合法的。

**规则 4：3-gram 复读惩罚**

在提取出的单词序列中，任何连续的三个单词构成一个 3-gram。如果该文档中存在任何一个 3-gram 出现次数严格大于 M 次，则判定为复读机文本，直接丢弃。例如：$M=1$ 时，序列 `["a", "b", "c", "a", "b", "c"]` 中，`("a","b","c")` 出现了 2 次，则应该丢弃。

**规则 5：规范化语义去重**

如果两篇文档提取出的单词序列完全一致（即忽略了标点符号和大小写的差异），判定为语义重复，仅保留按输入顺序第一次出现的那篇文档。

### 输入描述

第 1 行包含 5 个整数：`N L R M K`，分别代表文档数、最小长度、最大长度、3-gram 最大允许出现次数、黑名单词汇数。

第 2 行包含 K 个由空格分隔的黑名单词汇（保证全为小写）。如果 $K = 0$，则跳过此行，直接进入到文档内容。

接下来的 N 行，每行包含一篇原始文档（可能包含各种标点和多余空格）。

### 输出描述

按顺序输出所有通过清洗的高质量文档，每篇文档占一行。

### 样例 1

**输入**

```
3 10 100 2 1
spam
Buy cheap SPAM!!!
He is a spammer
He is, A! Spammer.
```

**输出**

```
He is a spammer
```

**说明**

- 第 1 行：文档数为 3，最小长度 10，最大长度 100，3-gram 最大允许出现次数 2，黑名单词汇数 1。
- 第 2 行：黑名单词汇为 `spam`。
- 文档 1：规范化后为 `Buy cheap SPAM!!!`，提取单词为 `["buy", "cheap", "spam"]`，精确命中黑名单 `spam`，丢弃。
- 文档 2：规范化后为 `He is a spammer.`，单词为 `["he", "is", "a", "spammer"]`，`spammer` 不触发 `spam` 拦截，保留。
- 文档 3：规范化后为 `He is, A! Spammer.`，单词为 `["he", "is", "a", "spammer"]`，单词序列与文档 2 完全相同，触发去重，丢弃。

### 样例 2

**输入**

```
2 10 200 2 0
a b c a b c a b c
a b c a b c
```

**输出**

```
a b c a b c
```

**说明**

- 第 1 行：文档数为 2，最小长度 10，最大长度 200，3-gram 最大允许出现次数 2，黑名单词汇数 0。
- 黑名单词汇数为 0，所以第 2 行开始为文档内容。
- 文档 1：3-gram `("a","b","c")` 出现了 3 次，大于阈值 2 次，丢弃。
- 文档 2：保留，输出该文档。

### 数据约束

$1 \le N \le 1000$，$1 \le L \le R \le 10000$，$0 \le M \le 100$，$0 \le K \le 100$。文档长度不超过 10000 字符。

## 解题思路
顺序执行五大清洗规则：规则1合并连续空白为单空格并去首尾空格，检查长度范围；规则2按非字母数字字符分割提取单词并转小写；规则3检查是否有独立单词精确匹配黑名单；规则4滑动窗口统计3-gram出现次数，超过阈值则丢弃；规则5用集合记录已出现的单词序列，实现语义去重（仅保留首次出现的文档）。全部通过后输出规范化字符串。

## 参考代码
```python
import sys
import re

def solve():
    lines = sys.stdin.read().splitlines()
    if not lines:
        return

    firstParts = lines[0].strip().split()
    N = int(firstParts[0])
    L = int(firstParts[1])
    R = int(firstParts[2])
    M = int(firstParts[3])
    K = int(firstParts[4])

    lineIdx = 1
    blacklist = set()
    if K > 0 and lineIdx < len(lines):
        blacklist = set(lines[lineIdx].strip().split())
        lineIdx += 1

    seenWordSeqs = set()

    for _ in range(N):
        if lineIdx >= len(lines):
            break
        doc = lines[lineIdx]
        lineIdx += 1

        # 规则1：空格规范化
        normalized = ' '.join(doc.split())
        if len(normalized) < L or len(normalized) > R:
            continue

        # 规则2：语义单词提取
        words = [w.lower() for w in re.split(r'[^a-zA-Z0-9]+', normalized) if w]
        if not words:
            continue

        # 规则3：黑名单拦截（独立单词精确匹配）
        skip = False
        for w in words:
            if w in blacklist:
                skip = True
                break
        if skip:
            continue

        # 规则4：3-gram复读惩罚
        trigramCount = {}
        for k in range(len(words) - 2):
            tg = (words[k], words[k + 1], words[k + 2])
            trigramCount[tg] = trigramCount.get(tg, 0) + 1
            if trigramCount[tg] > M:
                skip = True
                break
        if skip:
            continue

        # 规则5：语义去重
        wordSeq = tuple(words)
        if wordSeq in seenWordSeqs:
            continue
        seenWordSeqs.add(wordSeq)

        print(normalized)

if __name__ == '__main__':
    solve()
```
