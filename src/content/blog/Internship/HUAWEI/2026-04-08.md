### 1. 如果相关系数 $\rho_{xy}=-1$，说明 $X$ 和 $Y$：
- A. 相互独立
- B. 存在线性函数关系
- C. 完全负线性相关
- D. 完全不相关

:::fold[答案]

正确答案：C
解析：$\rho_{xy}=-1$ 表示 $X$ 和 $Y$ 之间是完全的负线性关系，即 $Y = aX + b$ 其中 $a < 0$。
:::

### 2. 对任意实矩阵 $A$，若其 SVD 为 $A=U\Sigma V^T$，则 $\text{rank}(A)$ 等于?
- A. $\Sigma$ 的迹 $\text{trace}(\Sigma)$
- B. $\Sigma$ 对角线上非零奇异值的个数
- C. $V$ 的行数
- D. $U$ 的列数

:::fold[答案]

正确答案：B
解析：矩阵的秩等于其非零奇异值的个数，即 $\Sigma$ 对角线上正奇异值的数量。
:::

### 3. $P(A)=0.4$，$P(B \mid A)=0.5$，$P(B \mid \neg A)=0.3$，则 $P(A \mid B)$ 最接近（）
- A. $0.4$
- B. $0.5$
- C. $0.3$
- D. $0.6$

:::fold[答案]

正确答案：B
解析：$P(B) = 0.5 \times 0.4 + 0.3 \times 0.6 = 0.38$，$P(A \mid B) = \frac{0.5 \times 0.4}{0.38} \approx 0.526$，最接近 $0.5$。

- B. 与RMSNorm相比主要区别在于去掉减去均值的部分
- C. LN计算每个样本的均值和方差→归一化→缩放加偏置

### 7. 损失函数曲面呈狭长山谷形状时SGD容易：

### 8. 关于KV Cache量化压缩，哪种策略是错误的？
- A. 在线量化，每次访问重新计算量化参数

- C. 分组量化（per-channel/per-head）平衡精度和效率
- D. 对 Key 和 Value 分别计算逐 token 缩放因子动态量化

## 第2题-路由器资源用量预测（P4729）| 中等 | 1000ms

### 题目描述

路由器的某资源利用率与多个运行特征强相关：协议连接数（单位：个）、转发数据包速率（单位：Mpps）、内存占用率（单位：%）。为了精准预测不同负载下的路由器资源利用率，保障网络稳定运行，请实现批量梯度下降法（BGD）来训练资源预测线性回归模型的参数。

**资源预测模型：**

$$y = w_0 + w_1 \cdot x_1 + w_2 \cdot x_2 + w_3 \cdot x_3$$

其中 $w_0$ 为偏置项，$w_1, w_2, w_3$ 为特征权重。

**损失函数：** 均方误差（MSE）

$$L = \frac{1}{2m} \sum_{i=1}^{m} (y_{\text{pred},i} - y_{\text{true},i})^2$$

其中 $m$ 为样本数。

**梯度更新规则：**

$$w_j = w_j - \alpha \cdot \frac{1}{m} \sum_{i=1}^{m} (y_{\text{pred},i} - y_{\text{true},i}) \cdot x_{ij}$$

其中偏置项 $w_0$ 对应 $x_{i0}=1$，$\alpha$ 为学习率。

**迭代规则：** 初始权重（含偏置）全为 $0$，迭代固定 $N$ 次后停止，无需判断收敛。

为了提高收敛速度，采用特征归一化进行训练，并在训练完成后进行权重还原：

**特征归一化：** 对每个特征维度 $x_j$（$j=1,2,3$）：

$$x_j^{\text{norm}} = \frac{x_j - \min(x_j)}{\max(x_j) - \min(x_j)}$$

其中 $\min(x_j)$ 为该特征在所有样本中的最小值，$\max(x_j)$ 为该特征在所有样本中的最大值，若 $\max(x_j)=\min(x_j)$（特征无波动），则归一化后的值为 $0$。

**特征权重还原：**

$$w_j = \frac{w_j^{\text{norm}}}{\max(x_j) - \min(x_j)} \quad (j = 1, 2, 3)$$

其中 $w_j^{\text{norm}}$ 为迭代后的权重，若 $\max(x_j) - \min(x_j)$ 为 $0$，$w_j$ 取 $0$。

**偏置项权重还原：**

$$w_0 = w_0^{\text{norm}} - \sum_{j=1}^{3} w_j \cdot \min(x_j)$$

其中 $w_0^{\text{norm}}$ 为迭代后偏置项的权重。

### 输入格式

- 第一行：整数 $m$（样本数量，$[1, 10000]$）
- 第二行：整数 $N$（迭代次数，$[1, 1000]$）
- 第三行：浮点数 $\alpha$（学习率，$[0, 1]$，保留 $2$ 位小数）
- 后续 $m$ 行：每行 $4$ 个整数，依次为 $x_1$（协议连接数，范围 $[0, 1000]$）、$x_2$（转发数据包速率，范围 $[0, 10000]$）、$x_3$（内存占用率，范围 $[0, 100]$）、$y$（资源用量，范围 $[0, 10000]$）

### 输出格式

一行，$4$ 个浮点数，依次为还原后的 $w_0, w_1, w_2, w_3$，结果保留 $2$ 位小数，银行家舍入，以一个空格分隔，前后无冗余空格。

### 样例1

**输入：**
```
3
100
0.10
100 200 150 6000
200 800 600 7500
300 70 60 6500
```

**输出：**
```
4394.59 6.82 1.20 1.55
```

**说明：**

样本数：$3$，迭代次数：$100$，学习率：$\alpha = 0.10$

样本 $1$：$100\ 200\ 150\ 6000$ 依次为 $x_1, x_2, x_3, y$

样本 $2$：$200\ 800\ 600\ 7500$

样本 $3$：$300\ 70\ 60\ 6500$

### 样例2

**输入：**
```
2
50
0.05
0 0 0 0
1000 10000 100 100000
```

**输出：**
```
11419.33 28.26 2.83 282.62
```

**说明：**

样本数：$2$，迭代次数：$50$，学习率：$\alpha = 0.05$

样本 $1$：$0\ 0\ 0\ 0$ 依次为 $x_1, x_2, x_3, y$

样本 $2$：$1000\ 10000\ 100\ 100000$

:::fold[答案]

## 解题思路
先对三个特征进行Min-Max归一化（处理特征无波动时归一化值为0）。初始化权重全为0，使用BGD迭代N次：每轮计算所有样本的预测误差，累加梯度后取均值更新权重。迭代完成后进行权重还原：特征权重除以对应的归一化缩放因子，偏置项减去各特征权重与最小值的点积。最终保留两位小数输出。

## 参考代码
```python
import sys


def solve():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    idx = 0
    m = int(data[idx]); idx += 1
    N = int(data[idx]); idx += 1
    alpha = float(data[idx]); idx += 1

    x1, x2, x3, y = [], [], [], []
    for _ in range(m):
        x1.append(float(data[idx])); x2.append(float(data[idx + 1]))
        x3.append(float(data[idx + 2])); y.append(float(data[idx + 3]))
        idx += 4

    # 特征归一化：Min-Max
    min1, max1 = min(x1), max(x1)
    min2, max2 = min(x2), max(x2)
    min3, max3 = min(x3), max(x3)

    range1 = max1 - min1
    range2 = max2 - min2
    range3 = max3 - min3

    x1n = [0.0] * m if range1 == 0 else [(v - min1) / range1 for v in x1]
    x2n = [0.0] * m if range2 == 0 else [(v - min2) / range2 for v in x2]
    x3n = [0.0] * m if range3 == 0 else [(v - min3) / range3 for v in x3]

    # 权重初始化
    w0, w1, w2, w3 = 0.0, 0.0, 0.0, 0.0

    # BGD迭代N次
    for _ in range(N):
        g0, g1, g2, g3 = 0.0, 0.0, 0.0, 0.0
        for i in range(m):
            pred = w0 + w1 * x1n[i] + w2 * x2n[i] + w3 * x3n[i]
            err = pred - y[i]
            g0 += err
            g1 += err * x1n[i]
            g2 += err * x2n[i]
            g3 += err * x3n[i]
        w0 -= alpha * g0 / m
        w1 -= alpha * g1 / m
        w2 -= alpha * g2 / m
        w3 -= alpha * g3 / m

    # 权重还原
    if range1 != 0:
        w1 = w1 / range1
    else:
        w1 = 0.0
    if range2 != 0:
        w2 = w2 / range2
    else:
        w2 = 0.0
    if range3 != 0:
        w3 = w3 / range3
    else:
        w3 = 0.0

    w0 = w0 - (w1 * min1 + w2 * min2 + w3 * min3)

    print(f"{w0:.2f} {w1:.2f} {w2:.2f} {w3:.2f}")


if __name__ == "__main__":
    solve()
```

## 第3题-快递员极速配送挑战（P4730）| 中等 | 2000ms

### 题目描述

某快递员负责一个片区的快递配送业务。假设他手头有 $N$ 个快递包裹需要派送，每个包裹对应一个具体的收货坐标 $(x_i, y_i)$（单位：公里）。

为了提高效率，公司要求快递员先利用聚类算法将这 $N$ 个包裹自动划分为 $K$ 个簇（代表 $K$ 个社区），快递员只需要将快递送到社区中心（类的中心点）即可。快递员从起始位置出发，按照每个社区中心与起点之间的距离由近到远排序，依次送完所有社区的快递，最后返回起始位置。已知快递员的平均行驶速度为 $speed$ km/h。

快递员初始坐标为 $(0, 0)$。请编写程序，计算完成所有配送并返回起点所需的总时间（单位：秒，向下取整）。

### K-Means 聚类计算步骤

**1. 种子点初始化**

- 将所有点按到起点的距离从小到大排序，如果距离相同的点，按照输入坐标点的先后顺序从小到大排序
- 选择排序后的前 $K$ 个点作为初始聚类中心

**2. 迭代优化**

- 将每个点 $p_i$ 分配到距离最近的聚类中心 $c_k$：

$$label_i = \arg\min_k dist(p_i, c_k)$$

- 重新计算每个聚类的中心点（$n_k$ 表示分配到第 $k$ 个聚类中心的坐标点个数），移动聚类中心：

$$c_k = \left( \frac{1}{n_k} \sum_{i}^{n_k} x_i,\ \frac{1}{n_k} \sum_{i}^{n_k} y_i \right)$$

**3. 收敛判断**

- 如果所有聚类中心的移动距离之和小于 $10^{-4}$，则停止迭代
- 如果达到了预设的最大迭代轮次，也停止迭代
- 否则返回第 $2$ 步继续进行迭代优化

### 输入格式

- 第一行输入 $3$ 个由空格分隔的整数，分别为 $K$（社区个数）、$N$（快递包裹总数）、$speed$（快递员平均行驶速度，单位 km/h）
- 接下来的 $N$ 行分别表示每个包裹的 $x$ 和 $y$ 坐标（单位：公里），用空格分割

### 输出格式

输出快递员送完所有快递所需的时间，保留整数，向下取整，单位 s。

### 样例2

**输入：**
```
5 3 10
1.00 1.00
2.00 2.00
3.00 3.00
```

**输出：**
```
3054
```

**说明：**

第一行输入：$5$ 个社区，$3$ 个包裹，快递员配送速度 $10$ km/h。

第二行及以后：$3$ 个快递包的具体坐标位置，x，y 坐标值，共有 $3$ 行，代表 $3$ 个包裹。

由于 $5 > 3$，$3$ 个包裹本身分别构成了 $3$ 个聚类中心，按照距离起点位置的排序配送的路径为：

$(0, 0) \rightarrow (1, 1) \rightarrow (2, 2) \rightarrow (3, 3) \rightarrow (0, 0)$

总距离为 $6\sqrt{2}$ km，配送速度为 $10$ km/h，总耗时为 $3600 \times 6\sqrt{2} \div 10 \approx 3054.70$ 秒，向下取整后为 $3054$ 秒，故输出 $3054$。

### 数据范围

- 使用欧式距离：$dist = \sqrt{(x_1 - x_2)^2 + (y_1 - y_2)^2}$
- 种子点的选择：对距离信息 $dist$ 进行升序排序，选择前 $K$ 个点进行初始化
- K-Means 终止条件：最大迭代轮次 $max\_iters = 50$，收敛容差 $tol = 10^{-4}$
- 用例数值范围：
  - $1 \le K \le 10$
  - $1 \le N \le 100$
  - $1 \le speed \le 100$

:::fold[答案]

## 解题思路
先按到原点的距离升序排序所有包裹（距离相同按输入顺序），取前K个作为初始聚类中心。使用标准KMeans迭代：分配点到最近聚类中心，重新计算聚类中心，直到聚类中心移动距离之和小于1e-4或达到50次迭代上限。聚类完成后将K个中心按到原点距离排序，配送路径为原点->近中心->远中心->原点，总距离除以速度再转换为秒并向下取整。

## 参考代码
```python
import sys
import math


def dist(p1, p2):
    return math.sqrt((p1[0] - p2[0]) ** 2 + (p1[1] - p2[1]) ** 2)


def solve():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    idx = 0
    K = int(data[idx]); idx += 1
    N = int(data[idx]); idx += 1
    speed = float(data[idx]); idx += 1

    points = []
    for i in range(N):
        x = float(data[idx]); y = float(data[idx + 1])
        idx += 2
        points.append((x, y, i))  # (x, y, 输入顺序)

    origin = (0.0, 0.0)

    # 种子点初始化：按到原点距离排序，距离相同按输入顺序
    sortedPts = sorted(points, key=lambda p: (dist(p, origin), p[2]))
    K = min(K, N)
    centroids = [(p[0], p[1]) for p in sortedPts[:K]]

    maxIters = 50
    tol = 1e-4

    for _ in range(maxIters):
        # 分配点到最近聚类中心
        clusters = [[] for _ in range(K)]
        for p in points:
            px, py = p[0], p[1]
            bestIdx = 0
            bestDist = float('inf')
            for ci, c in enumerate(centroids):
                d = math.sqrt((px - c[0]) ** 2 + (py - c[1]) ** 2)
                if d < bestDist:
                    bestDist = d
                    bestIdx = ci
            clusters[bestIdx].append(p)

        # 重新计算聚类中心
        newCentroids = []
        for ci in range(K):
            if clusters[ci]:
                n = len(clusters[ci])
                sx = sum(p[0] for p in clusters[ci])
                sy = sum(p[1] for p in clusters[ci])
                newCentroids.append((sx / n, sy / n))
            else:
                newCentroids.append(centroids[ci])

        # 检查收敛
        totalMove = 0.0
        for ci in range(K):
            totalMove += math.sqrt(
                (newCentroids[ci][0] - centroids[ci][0]) ** 2
                + (newCentroids[ci][1] - centroids[ci][1]) ** 2
            )
        centroids = newCentroids
        if totalMove < tol:
            break

    # 聚类中心按到原点距离排序
    centroids.sort(key=lambda c: dist(c, origin))

    # 计算总配送距离：原点 -> 各中心（由近到远） -> 原点
    totalDist = 0.0
    cur = origin
    for c in centroids:
        totalDist += dist(cur, c)
        cur = c
    totalDist += dist(cur, origin)

    # 时间（秒），向下取整
    timeSec = totalDist / speed * 3600.0
    print(int(timeSec))


if __name__ == "__main__":
    solve()
```
