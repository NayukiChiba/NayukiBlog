### 5. 设 $X$ 是非负整数的随机变量，则 $P(X \ge 1)$ 与 $E[X]$ 满足

## 第2题-网络流量分析（P4571）| 中等 | 1000ms

### 题目描述

网络流量分析是网络安全和性能优化的关键任务。假设你有一个包含网络流量数据的数据集，每条数据包含以下特征：

- $packet\_size$：数据包大小，单位，字节
- $inter\_arrival\_time$：数据包到达间隔时间，单位：毫秒
- $protocol\_type$：协议类型，如 $TCP、UDP、ICMP$ 等，已转换为数值编码

你的任务是使用 $KMeans$ 聚类算法对网络流量进行分类，分类后的中心点再经过一个分类头，能识别出可能的流量类型（如正常流量、异常流量、$DDoS$ 攻击流量等）。

$KMeans$ 算法原理为：先将数据分为 $K$ 组，随机选取 $K$ 个对象作为初始的聚类中心，然后计算每个对象与各个种子聚类中心之间的距离，将每一个对象分配给距离它最近的聚类中心，聚类中心以及分配给它们的对象就代表一个聚类。（为保证结果固定，本题初始的聚类中心已给出，不需要随机）

### 输入格式

- 初始聚类点个数 $k$
- 初始聚类中心集合（往下 $k$ 行，一行一个中心点）
- 迭代次数
- 样本个数 $m$
- 样本数 $m$，每个样本有三个特征（往下 $m$ 行，假定各特征数据已归一化处理，各维度权重占比一致）

### 输出格式

按给定次数迭代后新的聚类中心集合

### 样例1

**输入：**
```
3
50 25 30
60 15 60
25 75 90
3
9
50 25 30
30 50 30
60 15 60
25 75 90
10 05 60
26 15 30
32 67.5 90
80 7.5 60
20 100 90
```

**输出：**
```
35.33 30.00 30.00
50.00 9.17 60.00
25.67 80.83 90.00
```

**说明：**

采用欧式距离计算不同数据之间的距离：

$$
d=\sqrt{\left(x_{1}-x_{2}\right)^{2}+\left(y_{1}-y_{2}\right)^{2}+\left(z_{1}-z_{2}\right)^{2}}
$$

计算每个特征距离中心点的距离，选择距离最近的点作为当前特征归属的类别；按照划分的类别，使用重新计算每个类别的中心点，完成一轮迭代（中心点为对应类里所有样本的平均值）

重复上述流程 $k$ 次得到结果，最终结果保留两位小数，注意四舍五入

### 样例2

**输入：**
```
3
50 20 30
60 10 60
180 180 180
3
8
50 20 30
30 50 30
60 10 60
25 75 90
100 5 60
30 60 90

```

**输出：**
```

```

**说明：**

采用欧式距离计算不同数据之间的距离：

$$
d=\sqrt{\left(x_{1}-x_{2}\right)^{2}+\left(y_{1}-y_{2}\right)^{2}+\left(z_{1}-z_{2}\right)^{2}}
$$

计算每个特征距离中心点的距离，选择距离最近的点作为当前特征归属的类别；按照划分的类别，使用重新计算每个类别的中心点，完成一轮迭代（中心点为对应类里所有样本的平均值）

重复上述流程 $k$ 次得到结果，最终结果保留两位小数，注意四舍五入

### 数据范围

- 初始聚类中心点个数：$0 \le n \le 1000$，类型为 $int$
- 初始聚类中心点集合：每个特征的取值范围为 $0 \le f \le 1000$，类型为 $float$
- 迭代次数：$0 \le k \le 1000$，类型为 $int$
- 初始特征个数：$0 \le n \le 1000$，类型为 $int$
- 初始特征集合：每个特征的取值范围为 $0 \le f \le 1000$，类型为 $float$

:::fold[答案]

## 解题思路
标准KMeans聚类算法。使用给定的$K$个初始聚类中心，在每轮迭代中：将每个样本分配到距离最近的聚类中心（欧式距离），然后重新计算每个聚类的均值作为新的聚类中心。重复指定次数后输出最终聚类中心，保留两位小数。由于初始中心已给定且不随机，结果唯一确定。

## 参考代码
```python
import sys
import math


def solve():
    data = sys.stdin.read().strip().split()
    if not data:
        return
    idx = 0
    k = int(data[idx]); idx += 1

    # 读取初始聚类中心
    centroids = []
    for _ in range(k):
        x = float(data[idx]); y = float(data[idx + 1]); z = float(data[idx + 2])
        idx += 3
        centroids.append([x, y, z])

    iterations = int(data[idx]); idx += 1
    m = int(data[idx]); idx += 1

    # 读取样本点
    points = []
    for _ in range(m):
        x = float(data[idx]); y = float(data[idx + 1]); z = float(data[idx + 2])
        idx += 3
        points.append([x, y, z])

    # 执行指定次数的KMeans迭代
    for _ in range(iterations):
        clusters = [[] for _ in range(k)]
        # 分配每个点到最近的聚类中心
        for p in points:
            minDist = float('inf')
            minIdx = 0
            for ci, c in enumerate(centroids):
                d = math.sqrt((p[0] - c[0]) ** 2 + (p[1] - c[1]) ** 2 + (p[2] - c[2]) ** 2)
                if d < minDist:
                    minDist = d
                    minIdx = ci
            clusters[minIdx].append(p)

        # 重新计算聚类中心
        for ci in range(k):
            if clusters[ci]:
                n = len(clusters[ci])
                sumX = sum(p[0] for p in clusters[ci])
                sumY = sum(p[1] for p in clusters[ci])
                sumZ = sum(p[2] for p in clusters[ci])
                centroids[ci] = [sumX / n, sumY / n, sumZ / n]

    # 输出结果，保留两位小数
    for c in centroids:
        print(f"{c[0]:.2f} {c[1]:.2f} {c[2]:.2f}")


if __name__ == "__main__":
    solve()
```
:::

## 第3题-动态区间的多项式岭回归（P4572）| 困难 | 2000ms

### 题目描述

某大型互联网公司的数据中心，记录了其核心服务在连续 $N=200$ 天内的出口总流量（单位 $GB$，取值范围通常在 $100.00$ 到 $500.00$ 之间）已按时间顺序给出。由于监控系统维护，数据中有 $M$ 个（$M$ 的范围为 $20$ 到 $30$）缺失值，按顺序标记为 $Missing\_1, Missing\_2, ..., Missing\_M$。

已知这些缺失值保证不会出现在第 $1$ 天和最后 $1$ 天（即首尾两条记录一定存在）

**任务描述**

你需要为每一个缺失值，通过其邻近的、动态变化的真实数据区间，建立一个二阶多项式岭回归（$2nd$ $Ridge$ $Regression$）模型进行预测。

### 区间定义

对位于全局序号 $pos$ 的缺失值：

- **前向区间** $[left\_start, pos-1]$：从 $pos-1$ 开始向前（向着第 $1$ 天）寻找，遇到的第一个原始缺失值（$Missing\_1 ... Missing\_M$ 中的任意一个）的后一天。如果到第 $1$ 天仍未碰到任何原始缺失值，则前向区间的起始点 $left\_start$ 为第 $1$ 天。
- **后向区间** $[pos+1, right\_end]$：从 $pos+1$ 开始向后（向着第 $N$ 天）寻找，遇到的第一个原始缺失值的前一天。如果到第 $N$ 天仍未碰到任何原始缺失值，则后向区间的结束点 $right\_end$ 为第 $N$ 天。

取上述两个区间内的所有真实记录作为训练集 $(x, y)$，其中：

- $x$ 为日期序号（$1, 2, ..., N$）
- $y$ 为对应的流量值

### 模型

你需要使用岭回归求解二阶多项式模型：

$$
\hat{y} = \beta_2 x^2 + \beta_1 x + \beta_0
$$

岭回归的解可以通过以下矩阵公式计算：

$$
\beta = \left(X^{T} X + \lambda I \right)^{-1} X^{T} y
$$

其中：

- $\beta$ 是一个 $3 \times 1$ 的列向量，包含需要求解的系数 $[\beta_2, \beta_1, \beta_0]$
- $X$ 是一个 $n \times 3$ 的设计矩阵，其中 $n$ 是训练集的数据点数量。对于训练集中的每一个日期序号，矩阵 $X$ 中对应的一行为 $[x_i^{2}, x_i, 1]$
- $y$ 是一个 $n \times 1$ 的列向量，包含训练集中 $n$ 个观测点的流量值
- $X^{T}$ 是 $X$ 的转置矩阵
- $\lambda$ 是正则化参数，在本题中统一设为 $\lambda = 0.1$
- $I$ 是一个 $3 \times 3$ 的单位矩阵
- $(\cdot)^{-1}$ 表示矩阵求逆

### 输入格式

- 第 $1$ 行：两个整数 $M$ 和 $N$，由空格分隔。第一个参数 $M$ 指的是缺失值的总数（范围为 $20$ 到 $30$）；第二个参数 $N$ 是指后续的数据行数（固定为 $200$）
- 第 $2$ 行到第 $N+1$ 行：每行包含一个值，该值可以是：
  - 一个浮点数，代表当日的真实流量值
  - 一个字符串，格式为 $Missing\_i$（其中 $i$ 从 $1$ 到 $M$），代表当日的流量数据缺失

- 共 $M$ 行，严格按照 $Missing\_1, Missing\_2, ..., Missing\_M$ 的顺序输出
- 每行格式为 $Missing\_i$：xxx.xx，即标签、冒号、空格、预测值。预测值要求保留两位小数

## 解题思路
为每个缺失值动态确定训练区间：向前找到第一个原始缺失值的后一天作为区间左端点，向后找到第一个原始缺失值的前一天作为右端点。用区间内的真实数据构建二阶多项式岭回归（$\lambda = 0.1$），设计矩阵$X$包含$[x_i^2, x_i, 1]$，通过闭式解 $\beta = (X^\top X + \lambda I)^{-1} X^\top y$ 求解系数，再预测缺失位置的流量值。矩阵求逆采用$3 \times 3$伴随矩阵法手动实现，无需依赖numpy。

## 参考代码
```python
import sys


def matMul(A, B):
    """矩阵乘法 A(m×n) * B(n×p) -> (m×p)"""
    m, n = len(A), len(A[0])
    p = len(B[0])
    result = [[0.0] * p for _ in range(m)]
    for i in range(m):
        aik = A[i]
        for k in range(n):
            val = aik[k]
            if val != 0.0:
                bk = B[k]
                for j in range(p):
                    result[i][j] += val * bk[j]
    return result


def matTranspose(A):
    """矩阵转置"""
    m, n = len(A), len(A[0])
    return [[A[j][i] for j in range(m)] for i in range(n)]


def matInverse3x3(A):
    """3x3矩阵求逆（伴随矩阵法）"""
    a, b, c = A[0][0], A[0][1], A[0][2]
    d, e, f = A[1][0], A[1][1], A[1][2]
    g, h, i = A[2][0], A[2][1], A[2][2]

    det = a * (e * i - f * h) - b * (d * i - f * g) + c * (d * h - e * g)

    # 代数余子式矩阵
    A11 = e * i - f * h
    A12 = -(d * i - f * g)
    A13 = d * h - e * g
    A21 = -(b * i - c * h)
    A22 = a * i - c * g
    A23 = -(a * h - b * g)
    A31 = b * f - c * e
    A32 = -(a * f - c * d)
    A33 = a * e - b * d

    # 伴随矩阵 / 行列式
    return [
        [A11 / det, A21 / det, A31 / det],
        [A12 / det, A22 / det, A32 / det],
        [A13 / det, A23 / det, A33 / det],
    ]


def solve():
    lines = sys.stdin.read().strip().split('\n')
    if not lines:
        return

    firstLine = lines[0].split()
    M = int(firstLine[0])
    N = int(firstLine[1])

    # 1-indexed 存储
    values = [None] * (N + 1)
    missingPos = {}  # 位置 -> 缺失编号
    for i in range(1, N + 1):
        val = lines[i].strip()
        if val.startswith("Missing_"):
            num = int(val.split("_")[1])
            missingPos[i] = num
            values[i] = None
        else:
            values[i] = float(val)

    sortedMissing = sorted(missingPos.keys())
    results = {}
    lam = 0.1

    for pos in sortedMissing:
        # 前向区间：向前搜索第一个缺失值的后一天
        leftStart = 1
        for i in range(pos - 1, 0, -1):
            if values[i] is None:
                leftStart = i + 1
                break

        # 后向区间：向后搜索第一个缺失值的前一天
        rightEnd = N
        for i in range(pos + 1, N + 1):
            if values[i] is None:
                rightEnd = i - 1
                break

        # 收集训练数据
        trainX = []
        trainY = []
        for i in range(leftStart, pos):
            if values[i] is not None:
                trainX.append(i)
                trainY.append(values[i])
        for i in range(pos + 1, rightEnd + 1):
            if values[i] is not None:
                trainX.append(i)
                trainY.append(values[i])

        n = len(trainX)
        # 构建设计矩阵 X（n×3），每行 [x_i^2, x_i, 1]
        X = [[trainX[i] ** 2, float(trainX[i]), 1.0] for i in range(n)]
        y = [[trainY[i]] for i in range(n)]

        XT = matTranspose(X)
        XTX = matMul(XT, X)

        # X^T X + lambda * I
        for i in range(3):
            XTX[i][i] += lam

        XTX_inv = matInverse3x3(XTX)
        XTy = matMul(XT, y)
        beta = matMul(XTX_inv, XTy)

        beta2, beta1, beta0 = beta[0][0], beta[1][0], beta[2][0]
        pred = beta2 * pos * pos + beta1 * pos + beta0

        num = missingPos[pos]
        results[num] = f"Missing_{num}: {pred:.2f}"

    # 按 Missing_1 到 Missing_M 顺序输出
    for i in range(1, M + 1):
        print(results[i])


if __name__ == "__main__":
    solve()
```
