## 第1题-选择题（P4937）共20道，1-15单选，16-20多选

### 1. 在通信故障诊断RAG系统中，使用领域微调Embedding将查询与案例转换为768维向量。对向量执行 $L_2$ 归一化后，最优的相似度计算选择是（）
- A. 欧氏距离计算相似度，距离越小匹配度越高
- B. 点积相似度，结果与余弦相似度等价，且计算效率更高
- C. 曼哈顿距离，更适合高维稀疏故障文本向量场景
- D. 余弦相似度，需额外对向量模长进行归一化处理

:::fold[答案]

正确答案：B

解析：$L_2$ 归一化后向量模长为 1，此时点积 $\mathbf{u} \cdot \mathbf{v}$ 与余弦相似度 $\frac{\mathbf{u} \cdot \mathbf{v}}{\|\mathbf{u}\|\|\mathbf{v}\|}$ 数学等价，且点积计算更高效（省去除法操作）。
:::

:::

### 3. 当前主流多模态模型（LLaVA、Qwen-VL）与原生多模态模型（Gemini、GPT-4o）的核心架构差异是？
- A. 当前用Transformer，原生用CNN+RNN混合架构
- B. 当前冻结视觉编码器+训练投影层，原生端到端联合训练视觉和语言组件
- C. 当前参数量小，原生参数量大，这是唯一区别

正确答案：B

:::

### 4. SGD小批量梯度噪声ε对数值稳定性的影响，正确的是：
- A. 噪声可通过增大学习率完全消除
- B. 噪声方差与批量大小m成正比
- C. 噪声在接近最优解时帮助逃离鞍点，但可能导致在最优解附近波动
- D. 噪声导致目标函数值单调下降

:::fold[答案]

正确答案：C

解析：SGD 的梯度噪声来源于小批量采样的随机性。噪声方差与批量大小 $m$ 成反比（非正比），在接近最优解时噪声可帮助逃离鞍点，但也会导致在最优解附近持续波动，无法像全批量梯度下降那样精确收敛。
:::

### 5. 电影院150座位，第一人随机坐，后续人先坐自己座位（被占则随机选空位）。最后一人坐到自己座位的概率是？
- A. 113/150
- B. 49/150
- C. 77/150
- D. 1/2

:::fold[答案]

正确答案：D

解析：这是经典的"飞机座位问题"变体。利用递推或对称性分析可得：无论座位数 $n$ 为多少（$n \ge 2$），最后一人坐到自己座位的概率恒为 $1/2$，与总座位数 150 无关。
:::

### 6. 关于贪婪搜索（Greedy Search）的描述，正确的是：
- A. 每一步选择当前概率最高的Token，容易陷入局部最优或重复循环
- B. 通过引入随机性提高文本创造力
- C. 考虑所有可能序列路径，寻找全局最优解
- D. 计算量远大于束搜索（Beam Search）

:::fold[答案]

正确答案：A

解析：贪婪搜索每步仅选择概率最高的单个 Token，不考虑全局最优路径，计算量极小但容易陷入局部最优或产生重复循环。束搜索（Beam Search）维护多条候选路径，计算量反而更大。
:::

### 7. CLIP对比学习中batch size=N，正样本对(i,i)的梯度对图像特征 $z_i^I$ 的贡献包含：
- A. 仅对角线位置的梯度
- B. 仅图像→文本方向的正样本梯度
- C. 图像→文本的正样本梯度+作为其他样本负样本时的梯度
- D. 两个方向：图像→文本+文本→图像的正样本梯度

:::fold[答案]

正确答案：D

解析：CLIP 使用双向对比损失（InfoNCE），正样本对 $(i,i)$ 的梯度同时来自图像→文本和文本→图像两个方向：在图像→文本方向，$z_i^I$ 需拉近文本 $z_i^T$；在文本→图像方向，$z_i^I$ 需被文本 $z_i^T$ 匹配。两方向的梯度对 $z_i^I$ 均有贡献。
:::

### 8. DBSCAN算法相比K-Means的一个显著优点是能处理：

### 9. 自适应数值积分的基本策略是：

- B. 在估计误差较大的子区间上局部加密

### 12. Focal Loss通过引入哪种机制来调节不同样本的损失贡献？
- A. 动态权重衰减
- B. 调制因子(1-p_t)^γ，对已预测准确的简单样本降权
- C. 硬采样，丢弃简单样本只计算困难样本损失
- D. 标签平滑，将硬标签转换为软概率分布

正确答案：B

## 第2题-LLM 基石 - BPE 分词算法模拟（P4938）| 中等 | 1000ms

### 题目背景

某大型语言模型（LLM）无法直接理解文本，它需要将文本切分成一个个 Token，最常用的分词算法是 BPE（Byte Pair Encoding）。

BPE 的核心思想是：迭代合并最高频的相邻字符对。例如，给定初始字符串 `"hug hug"` 中，`"u"` 和 `"g"` 相邻出现了两次，频率最高，合并后为 `"h ug h ug"`。

### 任务

给定一个初始的字符串 $S$（由小写英文字母组成）和一个整数 $K$（最大合并次数），请模拟 BPE 算法的 $K$ 次迭代过程，输出最终的 Token 序列。

### 迭代规则

在每一轮迭代中：

1. **统计频率**：统计当前序列中所有相邻 Token 对的出现频率
2. **选择最优对**：找出频率最高的相邻对
3. **平局处理**：如果存在多个频率相同的最高频对，选择字典序最小的那一对。例如：`(a, b)` 比 `(c, d)` 小；`(a, b)` 比 `(a, c)` 小
4. **终止条件**：如果最高频率为 1（即所有相邻对都只出现一次），或者没有相邻对，则提前终止迭代
5. **合并**：将序列中所有出现的"最佳对"合并为一个新的 Token。合并时采用从左到右贪心原则（即遇到匹配就合并，然后跳过下一个，继续扫描）

### 输入描述

- 第一行：包含一个整数 $K$（$0 \le K \le 100$），表示最大合并次数
- 第二行：包含一个字符串 $S$，由小写英文字母组成，长度范围 $1 \le |S| \le 1000$

注意：初始状态下，将字符串 $S$ 中的每个字符视为一个独立的 Token。

### 输出描述

输出一行，表示经过最多 $K$ 次合并后的 Token 序列，Token 之间用空格分隔。

### 样例 1

**输入**

```
5
aaabdaaabac
```

**输出**

```
aaab d aaab a c
```

**说明**

初始 Token 序列：`a a a b d a a a b a c`

- Merge 1：`('a', 'a')` 频率最高（出现 4 次），合并后：`aa a b d aa a b a c`（贪心合并，前两个 `a` 合并，第三个落单）
- Merge 2：`('aa', 'a')` 频率最高（出现 2 次），合并后：`aab b d aaa b a c`
- Merge 3：`('aaa', 'b')` 频率最高（出现 2 次），合并后：`aaab d aaab a c`
- 后续所有相邻对频率均为 1，触发终止条件，停止迭代

### 样例 2

**输入**

```
2
ababac
```

**输出**

```
ab ab a c
```

**说明**

初始序列：`['a', 'b', 'a', 'b', 'a', 'c']`

- 第 1 轮：相邻对 `('a', 'b')` 出现 2 次，`('b', 'a')` 出现 2 次，`('a', 'c')` 出现 1 次。`('a', 'b')` 和 `('b', 'a')` 频率相同，按字典序选择 `('a', 'b')`。合并后序列：`['ab', 'ab', 'a', 'c']`
- 第 2 轮：所有相邻对频率均为 1，触发终止条件。最终输出：`ab ab a c`

:::fold[答案]
## 解题思路
模拟 BPE 迭代合并过程。每轮统计所有相邻 token 对的频率，选出频率最高且字典序最小的对，若最高频率为 1 则提前终止。合并时从左到右贪心扫描，匹配即合并然后跳过该对继续。关键点：合并后 token 变长，相邻对统计需基于合并后的新序列。

## 参考代码
```python
import sys

def solve():
    data = sys.stdin.read().splitlines()
    if not data:
        return
    K = int(data[0].strip())
    S = data[1].strip() if len(data) > 1 else ""
    # 初始 token 为单字符列表
    tokens = list(S)

    for _ in range(K):
        # 统计相邻对频率
        pairCount = {}
        for i in range(len(tokens) - 1):
            pair = (tokens[i], tokens[i + 1])
            pairCount[pair] = pairCount.get(pair, 0) + 1

        if not pairCount:
            break

        maxFreq = max(pairCount.values())
        if maxFreq <= 1:
            break

        # 选频率最高且字典序最小的对
        bestPair = min((p for p, f in pairCount.items() if f == maxFreq),
                       key=lambda x: (x[0], x[1]))

        # 贪心合并（从左到右）
        newTokens = []
        i = 0
        while i < len(tokens):
            if i + 1 < len(tokens) and tokens[i] == bestPair[0] and tokens[i + 1] == bestPair[1]:
                newTokens.append(bestPair[0] + bestPair[1])
                i += 2
            else:
                newTokens.append(tokens[i])
                i += 1
        tokens = newTokens

    sys.stdout.write(" ".join(tokens))

if __name__ == "__main__":
    solve()
```
:::

## 第3题-2D 单通道空洞卷积底层实现（P4939）| 中等 | 1000ms

### 题目背景

空洞卷积（膨胀卷积/Dilated Convolution）是卷积神经网络中扩大特征感受野且不增加参数/计算量的核心操作，广泛应用于语义分割等任务。本题要求基于原生语法/标准库手动实现单通道 2D 空洞卷积的底层逻辑，禁止调用任何卷积相关高级库函数。

### 计算逻辑详细步骤

**步骤 1：输入特征图的 Padding 填充**

对输入的 $H \times W$ 二维特征图进行常数 0 填充：在特征图的上下方向各填充 $padding$ 层 0，左右方向也各填充 $padding$ 层 0，得到尺寸为 $(H + 2 \times padding) \times (W + 2 \times padding)$ 的填充后特征图。

**步骤 2：计算有效卷积核尺寸与输出特征图尺寸**

有效卷积核尺寸（膨胀后的实际感受野尺寸）：设原始卷积核尺寸为 $K \times K$（$K$ 为奇数），膨胀率为 $dilation$（记为 $d$），则：

$$
K_{eff} = K + (K - 1) \times (d - 1)
$$

（$d = 1$ 时，$K_{eff} = K$，等价于普通卷积）

输出特征图尺寸：设输入特征图原始尺寸为 $H_{in} \times W_{in}$，步长为 $stride$（记为 $s$），填充为 $padding$（记为 $p$），则：

$$
H_{out} = \lfloor \frac{H_{in} + 2p - K_{eff}}{s} \rfloor + 1
$$

$$
W_{out} = \lfloor \frac{W_{in} + 2p - K_{eff}}{s} \rfloor + 1
$$

**步骤 3：滑动窗口遍历与空洞卷积核心计算**

- 窗口滑动规则：以步长 $stride$ 为间隔，在填充后的特征图上滑动有效尺寸为 $K_{eff}$ 的窗口，窗口总数对应输出特征图的 $H_{out} \times W_{out}$ 个位置
- 空洞卷积乘加规则：对每个滑动窗口，无需给卷积核插入空洞（0），而是按膨胀率 $dilation$ 间隔选取窗口内的元素与原始卷积核逐元素相乘：

设当前窗口在填充后特征图的起始坐标为 $(i_{start}, j_{start})$，遍历原始卷积核的每个位置 $(i_k, j_k)$（$0 \le i_k < K,\; 0 \le j_k < K$），计算该位置对应填充后特征图的坐标：

$$
i_{in} = i_{start} + i_k \times d
$$

$$
j_{in} = j_{start} + j_k \times d
$$

若 $(i_{in}, j_{in})$ 在填充后特征图的有效范围内，将填充后特征图的元素 $padded[i_{in}][j_{in}]$ 与卷积核元素 $kernel[i_k][j_k]$ 相乘，并累加到当前窗口的结果中。每个窗口的乘加累加结果作为输出特征图对应位置 $(i_{out}, j_{out})$ 的数值。

**步骤 4：输出结果**

将所有窗口的计算结果整理为 $H_{out} \times W_{out}$ 的二维矩阵，作为最终输出。

### 输入描述

- 输入特征图的大小（$H$ 和 $W$）
- 输入特征图的元素，共 $H \times W$ 个，全部为整数
- 卷积核的大小（$K$），为奇数
- 卷积核的元素，共 $K \times K$ 个，全部为整数
- 步长（$stride$）为 $\ge 1$ 的整数
- 填充（$padding$）为正整数（仅支持常数 0 填充，输入特征图上下左右各填充 $padding$ 层 0）
- 膨胀率（$dilation$）为 $\ge 1$ 的整数（$dilation = 1$ 时等价于普通 2D 卷积）

### 输出描述

逐行返回卷积后的二维特征图。

有效核尺寸 $K_{eff} = 3 + (3-1) \times (2-1) = 5$，输出尺寸 $H_{out} = (4 + 2 \times 1 - 5) / 1 + 1 = 2$，$W_{out} = (6 + 2 \times 1 - 5) / 1 + 1 = 4$。

填充后特征图尺寸为 $6 \times 8$，按膨胀率 2 采样（间隔 1 个元素），最终得到 $2 \times 4$ 的输出。

:::fold[答案]
## 解题思路
先对输入特征图进行零填充，再计算膨胀后的有效卷积核尺寸和输出尺寸。遍历每个输出位置，按膨胀率间隔采样窗口内元素，与原始卷积核逐元素乘加累加。核心公式：有效核尺寸 $K_{\text{eff}} = K + (K-1) \times (d-1)$，采样坐标偏移量为 $i_k \times d$。注意区分填充后的坐标与原始坐标的对应关系。

## 参考代码
```python
import sys

def solve():
    data = sys.stdin.read().split()
    if not data:
        return
    it = iter(data)
    H = int(next(it))
    W = int(next(it))
    total = H * W
    inputMat = []
    for _ in range(H):
        row = []
        for _ in range(W):
            row.append(int(next(it)))
        inputMat.append(row)

    K = int(next(it))
    kernel = []
    for _ in range(K):
        row = []
        for _ in range(K):
            row.append(int(next(it)))
        kernel.append(row)

    stride = int(next(it))
    padding = int(next(it))
    dilation = int(next(it))

    # 零填充
    paddedH = H + 2 * padding
    paddedW = W + 2 * padding
    padded = [[0] * paddedW for _ in range(paddedH)]
    for i in range(H):
        for j in range(W):
            padded[i + padding][j + padding] = inputMat[i][j]

    # 有效卷积核尺寸
    Keff = K + (K - 1) * (dilation - 1)

    # 输出尺寸
    Hout = (H + 2 * padding - Keff) // stride + 1
    Wout = (W + 2 * padding - Keff) // stride + 1

    output = [[0] * Wout for _ in range(Hout)]

    for oi in range(Hout):
        for oj in range(Wout):
            iStart = oi * stride
            jStart = oj * stride
            acc = 0
            for ki in range(K):
                for kj in range(K):
                    pi = iStart + ki * dilation
                    pj = jStart + kj * dilation
                    if 0 <= pi < paddedH and 0 <= pj < paddedW:
                        acc += padded[pi][pj] * kernel[ki][kj]
            output[oi][oj] = acc

    # 输出结果
    outLines = []
    for row in output:
        outLines.append(" ".join(str(v) for v in row))
    sys.stdout.write("\n".join(outLines))

if __name__ == "__main__":
    solve()
```
