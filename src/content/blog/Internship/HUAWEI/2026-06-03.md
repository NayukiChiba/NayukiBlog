## 第1题-选择题（P4996）

> 共 20 道选择题，其中 1-15 为单选，16-20 为多选。

### 1. CUDA 中的双缓冲（Double Buffering）技术主要用于解决什么问题？

- A. 减少显存占用
- B. 掩盖访存延迟（Hide Latency）
- C. 避免死锁
- D. 提高计算精度

:::fold[答案]

正确答案：B

解析：双缓冲使用两个缓冲区交替工作，一个用于计算的同时另一个进行数据传输，通过计算与数据传输的重叠来隐藏访存延迟，提升整体吞吐量。
:::

### 2. 考虑一个深层神经网络，所有隐藏层都使用 ReLU 激活函数。在训练过程中，发现某些神经元的输出始终为 0，且梯度也为 0，这种现象被称为？

- A. 梯度消失
- B. 神经元死亡
- C. 梯度爆炸
- D. 过拟合

:::fold[答案]

正确答案：B

解析：ReLU 在输入为负时输出 0、梯度为 0，若神经元因权重更新进入恒负激活区域，将永远无法恢复，称为"死亡 ReLU"或神经元死亡。梯度消失是指梯度随层数增加呈指数衰减，不等同于神经元死亡。
:::

### 3. 在某些算子融合优化的底层实现中，需要通过迭代法快速计算平方根。设目标是求 $\sqrt{3}$ 的近似值，将其转化为求函数 $f(x)=x^2-3=0$ 的正根。若使用牛顿迭代法，且选取初始点 $x_0=2$，则单次迭代后的值 $x_1$ 为

- A. $1.75$
- B. $1.8$
- C. $1.667$
- D. $1.5$

:::fold[答案]

正确答案：A

解析：牛顿迭代公式 $x_{n+1} = x_n - \frac{f(x_n)}{f'(x_n)} = x_n - \frac{x_n^2 - 3}{2x_n}$，代入 $x_0 = 2$ 得 $x_1 = 2 - \frac{4-3}{4} = 1.75$。
:::

### 4. 在一个垃圾文本自动过滤系统中，基于朴素贝叶斯模型，系统提取了文本中的两个特征词：$W1$和$W2$。已知 $P(Spam)=0.4$，垃圾文本中包含$W1$的概率为0.5，包含$W2$的概率为0.6；正常文本中包含$W1$的概率为0.1，包含$W2$的概率为0.2。假设特征在类条件独立，一段同时包含$W1$和$W2$的文本是垃圾文本的概率最接近？

- A. $0.5$
- B. $0.83$
- C. $0.75$
- D. $0.91$

:::fold[答案]

正确答案：D

解析：$P(\text{Spam} \mid W_1, W_2) = \frac{0.5 \times 0.6 \times 0.4}{0.5 \times 0.6 \times 0.4 + 0.1 \times 0.2 \times 0.6} = \frac{0.12}{0.132} \approx 0.91$。
:::

### 5. 关于 Adam 中偏置校正（bias correction）的作用，下列说法最准确的是哪一项？

- A. 用于防止梯度爆炸，因此可以替代梯度裁剪
- B. 用于修正一阶矩和二阶矩在训练初期因初始化为 $0$ 而产生的估计偏小问题
- C. 用于让 Adam 退化成 SGD，从而提高泛化能力
- D. 用于减少模型参数数量，提高推理速度

:::fold[答案]

正确答案：B

解析：Adam 的一阶矩 $m_t$ 和二阶矩 $v_t$ 初始化为 0，早期迭代中指数移动平均偏向于 0，导致估计偏小。偏置校正通过除以 $(1 - \beta^t)$ 修正这一偏差，使早期估计更准确。
:::

### 6. 以下哪项不是 Transformer 优于传统 RNN 的典型特征：

- A. 推理时的单步解码时间复杂度更低
- B. 更高的训练并行度
- C. 更强的长距离依赖建模能力
- D. 训练层数较深的模型时更好地避免梯度消失问题

:::fold[答案]

正确答案：A

解析：Transformer 推理时每步需计算当前 token 对所有历史 token 的注意力，复杂度为 $O(n)$（带 KV Cache），而 RNN 仅需 $O(1)$ 的单步状态更新。B、C、D 均为 Transformer 的真正优势。
:::

### 7. 下列哪项不是量化的目的？

- A. 减小模型体积
- B. 提升推理吞吐量
- C. 提高训练收敛速度
- D. 适配移动端硬件

:::fold[答案]

正确答案：C

解析：量化（INT8/FP8/INT4）通过降低数值精度来减小模型体积（A）、加速推理（B）并适配资源受限的移动端（D）。量化训练（QAT）通常不会加快收敛，反而可能因精度降低而需要更多训练步数。
:::

### 8. 在大模型 KV Cache 压缩技术中，若利用二次多项式拟合一段 Token 的 Attention Score 趋势，其范德蒙德矩阵（Vandermonde Matrix）的条件数（Condition Number）随节点增加而迅速增大，这会导致？

- A. 拟合残差（Residual）变为 $0$
- B. 硬件加速器的吞吐量线性提升
- C. 自动微分失效
- D. 计算结果的数值稳定性变差

:::fold[答案]

正确答案：D

解析：范德蒙德矩阵的条件数随节点数指数增长，使得线性方程组求解对输入扰动极为敏感，导致拟合系数的数值稳定性变差、结果不可靠。这是多项式拟合中著名的病态问题。
:::

### 9. 交叉熵损失函数通常与哪个激活函数配合使用

- A. Tanh
- B. Softmax
- C. Sigmoid
- D. ReLU

:::fold[答案]

正确答案：B

解析：交叉熵损失与 Softmax 是分类任务的标准组合，Softmax 将 logits 转换为概率分布，交叉熵度量预测分布与真实分布的差异。两者配合使用时梯度形式简洁（$\hat{y} - y$），数值稳定。
:::

### 10. 对于类别不平衡的分类任务，哪种损失函数可以缓解不平衡问题

- A. 交叉熵损失
- B. Focal Loss
- C. Hinge 损失
- D. 均方误差

:::fold[答案]

正确答案：B

解析：Focal Loss 在交叉熵基础上引入调制因子 $(1-p_t)^{\gamma}$，降低易分类样本的权重，使模型聚焦于难分类的少数类样本，专为解决类别不平衡问题设计。
:::

### 11. 在智能体（Agent）的记忆机制中，短期记忆用于存储当前对话上下文，长期记忆用于存储用户偏好、历史任务记录等持久化信息。某客服 Agent 在多轮对话中无法识别用户重复提出的"网络延迟"问题，下列改进方案最合理的是（ ）

- A. 将用户关于"网络延迟"的关键信息结构化存入长期记忆（如记录用户的常见问题或持续性故障），并为该类条目设计基于语义检索的触发策略，使 Agent 能在后续对话中自动联想并引用这些长期记忆
- B. 调整对话轮次截断策略，适当增大短期记忆窗口，并在每轮对话中附最近若干轮的摘要信息，以减少"网络延迟"相关内容被模型遗忘的概率
- C. 引入专门的"网络质量排障"工具，并在用户提及"网络慢""卡顿"等表述时优先触发该工具，以便通过工具调用结果来辅助模型识别并处理网络延迟问题
- D. 优化提示词（Prompt），在系统提示中显式强调"关注用户反复提及的问题"，并增加对"网络延迟"等关键词的示例和说明，以提升大模型对该问题类型的敏感度

:::fold[答案]

正确答案：A

解析：问题的核心是 Agent 在跨对话轮次中无法"记住"用户曾提及的问题。长期记忆 + 语义检索触发是解决跨对话信息持久化的根本方案。B 仅扩大短期窗口无法跨对话持久化；C 和 D 是单轮内的改进，不解决记忆机制缺失的问题。
:::

### 12. RNN 在处理长序列时的主要问题是

- A. 梯度消失 / 爆炸
- B. 计算复杂度高
- C. 内存占用大
- D. 参数过多

:::fold[答案]

正确答案：A

解析：RNN 通过时间反向传播（BPTT），梯度在长序列上反复乘以相同的权重矩阵，若特征值绝对值偏离 1，梯度会呈指数级衰减（消失）或增长（爆炸），导致长距离依赖无法有效学习。
:::

### 13. 随机变量 $X$ 的概率分布为：$P(X=1)=0.2, P(X=2)=0.5, P(X=3)=0.3$。它的期望 $E(X)$ 是？

- A. $2.1$
- B. $2.0$
- C. $1.8$
- D. $2.5$

:::fold[答案]

正确答案：A

解析：$E(X) = 1 \times 0.2 + 2 \times 0.5 + 3 \times 0.3 = 0.2 + 1.0 + 0.9 = 2.1$。
:::

### 14. 某服务台处理请求的时间服从指数分布。已知处理某个请求已经花去了 10 分钟还没结束，它还需要再处理至少 5 分钟的概率，与一个全新请求需要处理至少 5 分钟的概率相比？

- A. 无法比较
- B. 两者相等
- C. 前者大于后者
- D. 前者小于后者

:::fold[答案]

正确答案：B

解析：指数分布具有无记忆性：$P(X > t + s \mid X > s) = P(X > t)$。无论已过去多久，剩余等待时间的分布与从头开始相同，因此两者概率相等。
:::

### 15. 所谓的"死亡神经元"现象，在反向传播视角看，是指？

- A. 该神经元的偏置项过大
- B. 该神经元的激活函数爆炸
- C. 该神经元的输出恒为 0，且无论输入如何，梯度恒为 0，权重永远不再更新
- D. 该神经元的权重变成了 NaN

:::fold[答案]

正确答案：C

解析：ReLU 在输入为负时输出和梯度均为 0，反向传播时该神经元的权重不接收任何梯度更新，造成永久性"死亡"。这就是第 2 题描述的同一现象，本题从反向传播视角强调梯度恒为 0 的后果。
:::

### 16.（多选）激活函数的选择需要考虑哪些因素

- B. 计算资源限制
- C. 网络的深度
- D. 任务类型（分类 / 回归）

:::fold[答案]

正确答案：A, B, C, D

解析：全部四项均影响激活函数选择。数据特点影响激活函数的适用性（如有无负值）；计算资源影响选择复杂度（如 Swish 比 ReLU 开销大）；网络深度影响梯度流动（深层网络需避免饱和激活函数）；任务类型决定输出层激活函数（分类用 Softmax/Sigmoid，回归用线性）。
:::

### 17.（多选）关于截断误差和舍入误差的区别，下列说法正确的有（ ）

- A. 截断误差是由"近似替代"产生的，舍入误差是由"有限精度运算"产生的
- B. 截断误差只存在于无穷级数近似中，舍入误差只存在于小数运算中
- C. 截断误差是系统性误差，舍入误差是随机性误差

## 第2题-智能物流定价引擎（在线学习）（P4997）

### 题目描述

你在一家同城物流平台负责"运费估计引擎"。

平台记录了大量历史订单，每条订单有两个关键字段：

- $x$：运输距离（km）
- $y$：成交运费（元）

为了给新订单快速报价，系统使用一元线性回归模型：

$$y = a * x + b$$

但市场价格持续波动，历史数据会不断新增。你需要实现一个在线学习模块，按操作实时维护模型并回答预测请求。

### 操作定义

- **ADD** $x$ $y$：新增一条历史样本 $(x,y)$
- **QUERY** $x_0$：基于当前所有样本拟合最小二乘直线，并输出在 $x_0$ 处的预测值

### 模型定义

在任意一次 QUERY 时，设当前样本集合为 $(x_i, y_i)$，模型参数 $(a,b)$ 必须满足：使平方误差和 $\sum(y_i - (a * x_i + b))^2$ 最小。也就是说，QUERY $x_0$ 输出的应是该最优模型在 $x_0$ 处的预测值。

对于一元线性回归，若当前样本数为 $n$，并记：

- $Sx = \sum x_i$
- $Sy = \sum y_i$
- $Sxx = \sum x_i^2$
- $Sxy = \sum x_i * y_i$

当最优解唯一时，可直接使用最小二乘法公式：

$$a = (n * Sxy - Sx * Sy) / (n * Sxx - Sx^2)$$

$$b = (Sy - a * Sx) / n$$

于是 QUERY $x_0$ 的答案为：

$$yhat = a * x_0 + b$$

### 退化规则

- 若 $n = 0$，QUERY 输出 $0.000000$
- 若 $n \ge 1$ 且最优解不唯一（等价于所有 $x$ 相同），为保证答案唯一，规定使用常数模型：
  - $a = 0$
  - $b =$ 当前所有 $y$ 的平均值
  - $yhat = b$

### 数据范围

- $1 \le Q \le 200000$
- $-10^6 \le x, y, x_0 \le 10^6$
- 保证中间统计量在 64 位有符号整数范围内

### 输入描述

- 第 1 行：整数 $Q$，表示操作总数
- 接下来 $Q$ 行，每行为以下两种之一：
  - `ADD x y`
  - `QUERY x0`

### 输出描述

对每个 QUERY 输出一行预测值，结果必须输出为固定 6 位小数。

### 样例1

**输入：**
```
8
ADD 10 35
ADD 20 55
QUERY 15
ADD 30 78
QUERY 25
ADD 25 69
QUERY 40
QUERY 5
```

**输出：**
```
45.000000
66.750000
100.285714
23.685714
```

**说明：** 前两条样本后，回归线为 $y = 2x + 15$，所以 QUERY 15 输出 45.000000。加入更多样本后，参数会变化，因此后续查询结果也会变化。

### 样例2

**输入：**
```
7
ADD 12 30
ADD 12 36
QUERY 12
ADD 12 45
QUERY 100
QUERY -20
QUERY 0
```

**输出：**
```
33.000000
37.000000
37.000000
37.000000
```

**说明：** 所有样本的 $x$ 都等于 12，最小二乘解不唯一。按退化规则使用常数模型，预测值恒为当前 $y$ 的平均值。

:::fold[答案]
## 解题思路
在线维护累积统计量 Sx, Sy, Sxx, Sxy 及样本计数 n。QUERY 时：n=0 输出 0；若所有 x 相同（即 n*Sxx == Sx^2）则使用常数模型 a=0, b=平均 y；否则用最小二乘公式计算 a 和 b，代入 x0 得预测值。所有运算使用整数或浮点，输出保留 6 位小数。注意 64 位有符号整数范围内保证精度。

## 参考代码
```python
import sys

def solve():
    data = sys.stdin.read().splitlines()
    if not data:
        return
    Q = int(data[0].strip())

    n = 0
    Sx = 0
    Sy = 0
    Sxx = 0
    Sxy = 0

    outLines = []

    for lineIdx in range(1, Q + 1):
        parts = data[lineIdx].split()
        op = parts[0]

        if op == "ADD":
            x = int(parts[1])
            y = int(parts[2])
            n += 1
            Sx += x
            Sy += y
            Sxx += x * x
            Sxy += x * y

        elif op == "QUERY":
            x0 = int(parts[1])
            if n == 0:
                yhat = 0.0
            elif n * Sxx == Sx * Sx:
                # 所有 x 相同，退化规则
                b = Sy / n
                yhat = b
            else:
                a = (n * Sxy - Sx * Sy) / (n * Sxx - Sx * Sx)
                b = (Sy - a * Sx) / n
                yhat = a * x0 + b
            outLines.append(f"{yhat:.6f}")

    sys.stdout.write("\n".join(outLines))

if __name__ == "__main__":
    solve()
```

## 第3题-回归任务反向传播机理（P4998）

### 题目描述

在深度学习的训练过程中，反向传播（Backpropagation）是计算梯度并更新网络参数的核心算法。本题要求实现一个用于回归任务的三层 MLP（多层感知机）的单次前向传播与反向传播，给定网络参数、一个输入样本及其目标值，计算梯度下降更新后的新参数。

### 网络架构与参数

网络为三层全连接结构：

输入层 $(d_{in})$ → 隐藏层 $(d_{hidden}, ReLU)$ → 输出层 $(d_{out}, 线性)$

其中 $d_{in}$、$d_{hidden}$、$d_{out}$ 分别为各层的维度（神经元个数），网络中所有向量和矩阵的形状均由此三个维度唯一确定。各参数和中间变量定义如下：

| 符号 | 形状 | 说明 |
|---|---|---|
| $x$ | $(d_{in})$ | 输入特征向量 |
| $W_1$ | $(d_{hidden} \times d_{in})$ | 输入层到隐藏层的权重矩阵，第 $i$ 行对应隐藏层第 $i$ 个神经元 |
| $b_1$ | $(d_{hidden})$ | 隐藏层偏置向量 |
| $z_1$ | | 隐藏层预激活值，$z_1 = W_1 x + b_1$ |
| $a_1$ | | 隐藏层激活值，$a_1 = ReLU(z_1)$ |
| $W_2$ | $(d_{out} \times d_{hidden})$ | 隐藏层到输出层的权重矩阵，第 $i$ 行对应输出层第 $i$ 个神经元 |
| $b_2$ | $(d_{out})$ | 输出层偏置向量 |
| $z_2 / y_{pred}$ | | 输出层预激活值即为预测值（线性激活），$z_2 = W_2 a_1 + b_2$ |
| $y_{true}$ | | 目标真实值 |
| $\eta$ | 标量 | 学习率 |

### 计算流程

**1. 前向传播**

$$z_1 = W_1 x + b_1$$
$$a_1 = ReLU(z_1)$$
$$y_{pred} = W_2 a_1 + b_2$$

**2. 损失函数**

$$L = \frac{1}{2} \sum_{i=0}^{d_{out}-1} (y_{pred,i} - y_{true,i})^2$$

**3. 反向传播**

输出层梯度：

$$\delta_2 = y_{pred} - y_{true}$$
$$\nabla W_2 = \delta_2 \otimes a_1 \quad (\text{外积，结果形状 } d_{out} \times d_{hidden})$$
$$\nabla b_2 = \delta_2$$

隐藏层梯度：

$$\delta_1 = (W_2^\top \delta_2) \odot ReLU'(z_1)$$
$$\nabla W_1 = \delta_1 \otimes x \quad (\text{外积，结果形状 } d_{hidden} \times d_{in})$$
$$\nabla b_1 = \delta_1$$

其中 $\otimes$ 为向量外积，$\odot$ 为逐元素乘法，$ReLU'(z)$ 在 $z = 0$ 时为 $0$。

**4. 权重更新（梯度下降）**

$$W_{new} = W - \eta \cdot \nabla W, \quad b_{new} = b - \eta \cdot \nabla b$$

对 $W_1, b_1, W_2, b_2$ 分别执行上述更新。

### 数据约束

- $1 \le d_{in}, d_{hidden}, d_{out} \le 10$
- 权重和偏置的初始值绝对值不超过 $5$
- 输入特征和目标值绝对值不超过 $10$
- $0.001 \le \eta \le 1.0$

### 输入描述

- 一行：$d_{in}$ $d_{hidden}$ $d_{out}$
- $W_1$：共 $d_{hidden}$ 行，每行 $d_{in}$ 个浮点数
- $b_1$：一行，$d_{hidden}$ 个浮点数
- $W_2$：共 $d_{out}$ 行，每行 $d_{hidden}$ 个浮点数
- $b_2$：一行，$d_{out}$ 个浮点数
- $x$：一行，$d_{in}$ 个浮点数
- $y_{true}$：一行，$d_{out}$ 个浮点数
- $\eta$：一个浮点数

### 输出描述

- $W_{1new}$：共 $d_{hidden}$ 行，每行 $d_{in}$ 个浮点数
- $b_{1new}$：一行，$d_{hidden}$ 个浮点数
- $W_{2new}$：共 $d_{out}$ 行，每行 $d_{hidden}$ 个浮点数
- $b_{2new}$：一行，$d_{out}$ 个浮点数
- $y_{pred}$：一行，$d_{out}$ 个浮点数

所有浮点数保留 6 位小数，同一行以内以空格分隔。

### 样例1

**输入：**
```
2 2 2
0.4 0.3
-0.5 0.6
0.1 -0.2
0.5 -0.3
0.2 0.4
0.0 0.1
1.0 0.5

```

**输出：**

```
0.120300 -0.200000
0.522750 -0.300000
0.209100 0.400000
0.035000 0.114000
0.325000 0.230000
```

**说明：** 本例网络为 $2 \to 2 \to 2$ 的多输出结构。

前向传播时，$z_1 = [0.65, -0.4]$，第 2 个神经元被 ReLU 关闭，$a_1 = [0.65, 0]$。

预测值 $y_{pred} = [0.325, 0.23]$。

反向传播时，$\delta_2 = [-0.175, -0.07]$ 是二维向量，两个输出分量的误差通过 $W_2^\top \delta_2$ 叠加后传回隐藏层；

但第 2 个神经元被 ReLU 阻断，因此只有第 1 个神经元对应的 $W_1$ 第 1 行和 $b_1$ 第 1 个分量发生更新，第 2 行保持不变。

### 样例2

**说明：** 本例网络为 $2 \to 2 \to 1$ 结构。

前向传播时，$z_1 = [-2.5, -1.6]$ 的两个分量均为负值，ReLU 将它们全部置零（$a_1 = [0, 0]$），预测值完全由输出层偏置决定：$y_{pred} = b_2 = 0.1$。

反向传播时，由于 $a_1$ 全为零且 $ReLU'(z_1) = [0, 0]$，所有流回隐藏层的梯度均被完全阻断，导致 $W_1$、$b_1$、$W_2$ 均未更新，只有 $b_2$ 因直接连接输出层而从 $0.1$ 更新为 $0.14$。

:::fold[答案]
## 解题思路
严格按照题目公式逐步计算。前向传播：z1 = W1*x + b1，a1 = ReLU(z1)，ypred = W2*a1 + b2。反向传播：delta2 = ypred - ytrue，grad_W2 = delta2 外积 a1，grad_b2 = delta2；delta1 = (W2^T * delta2) * ReLU'(z1)，grad_W1 = delta1 外积 x，grad_b1 = delta1。梯度下降更新所有参数。注意 ReLU 导数在 z=0 时取 0，输出保留 6 位小数。

## 参考代码
```python
import sys

def relu(z):
    return max(0.0, z)

def reluDerivative(z):
    return 0.0 if z <= 0 else 1.0

def vectorDot(a, b):
    """向量点积"""
    return sum(ai * bi for ai, bi in zip(a, b))

def matrixVectorMul(mat, vec):
    """矩阵乘向量: mat (R x C), vec (C,) -> (R,)"""
    return [sum(mat[i][j] * vec[j] for j in range(len(vec))) for i in range(len(mat))]

def vectorOuter(a, b):
    """向量外积: a (R,), b (C,) -> (R x C)"""
    return [[ai * bj for bj in b] for ai in a]

def matrixTransposeMulVec(mat, vec):
    """mat^T * vec: mat (R x C), vec (R,) -> (C,)"""
    R = len(mat)
    C = len(mat[0])
    res = [0.0] * C
    for j in range(C):
        s = 0.0
        for i in range(R):
            s += mat[i][j] * vec[i]
        res[j] = s
    return res

def solve():
    data = sys.stdin.read().split()
    if not data:
        return
    it = iter(data)

    din = int(next(it))
    dhidden = int(next(it))
    dout = int(next(it))

    # W1: dhidden x din
    W1 = [[float(next(it)) for _ in range(din)] for _ in range(dhidden)]
    # b1: dhidden
    b1 = [float(next(it)) for _ in range(dhidden)]
    # W2: dout x dhidden
    W2 = [[float(next(it)) for _ in range(dhidden)] for _ in range(dout)]
    # b2: dout
    b2 = [float(next(it)) for _ in range(dout)]
    # x: din
    x = [float(next(it)) for _ in range(din)]
    # ytrue: dout
    ytrue = [float(next(it)) for _ in range(dout)]
    # eta
    eta = float(next(it))

    # ---- 前向传播 ----
    z1 = [sum(W1[i][j] * x[j] for j in range(din)) + b1[i] for i in range(dhidden)]
    a1 = [relu(v) for v in z1]
    ypred = [sum(W2[i][j] * a1[j] for j in range(dhidden)) + b2[i] for i in range(dout)]

    # ---- 反向传播 ----
    delta2 = [ypred[i] - ytrue[i] for i in range(dout)]

    # grad_W2 = delta2 外积 a1  (dout x dhidden)
    gradW2 = [[delta2[i] * a1[j] for j in range(dhidden)] for i in range(dout)]
    gradB2 = list(delta2)

    # ReLU 导数
    reluPrime = [reluDerivative(v) for v in z1]

    # delta1 = (W2^T * delta2) * relu'(z1)  逐元素乘
    w2tDelta2 = matrixTransposeMulVec(W2, delta2)  # dhidden
    delta1 = [w2tDelta2[i] * reluPrime[i] for i in range(dhidden)]

    # grad_W1 = delta1 外积 x  (dhidden x din)
    gradW1 = [[delta1[i] * x[j] for j in range(din)] for i in range(dhidden)]
    gradB1 = list(delta1)

    # ---- 梯度下降更新 ----
    W1new = [[W1[i][j] - eta * gradW1[i][j] for j in range(din)] for i in range(dhidden)]
    b1new = [b1[i] - eta * gradB1[i] for i in range(dhidden)]
    W2new = [[W2[i][j] - eta * gradW2[i][j] for j in range(dhidden)] for i in range(dout)]
    b2new = [b2[i] - eta * gradB2[i] for i in range(dout)]

    # ---- 输出 ----
    outLines = []
    for row in W1new:
        outLines.append(" ".join(f"{v:.6f}" for v in row))
    outLines.append(" ".join(f"{v:.6f}" for v in b1new))
    for row in W2new:
        outLines.append(" ".join(f"{v:.6f}" for v in row))
    outLines.append(" ".join(f"{v:.6f}" for v in b2new))
    outLines.append(" ".join(f"{v:.6f}" for v in ypred))

    sys.stdout.write("\n".join(outLines))

if __name__ == "__main__":
    solve()
```
