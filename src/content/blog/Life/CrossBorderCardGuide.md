## 前言

出国旅行或留学时，携带一张支持外币支付的银行卡至关重要。**跨境通信记卡**（通常卡面带有 VISA 或 MasterCard 标识）是中国银行推出的一类银联借记卡，支持 19 种外币存储与消费，能让你在境外刷卡、取现更加便捷。

本文将详细介绍跨境通信记卡的完整使用流程与注意事项，建议**出国前提前准备**，避免临行前手忙脚乱。

---

## 一、准备工作

### 1.1 开立一类银联借记卡

首先需要拥有一张**一类银联借记卡**，并开通手机银行。

> **警告**:  **注意**：二类卡无法进行购汇操作。如果你的卡是二类卡，需前往银行网点升级为一类卡。

### 1.2 开立跨境通信记卡

携带身份证前往银行柜台，申请开立跨境通信记卡。卡面通常带有 **VISA** 或 **MasterCard（万事达）** 标识。

### 1.3 关联手机银行

登录手机银行 App，在首页进入「**账户管理**」，将新开的跨境通信记卡关联到手机银行中，方便后续操作。

---

## 二、核心功能设置

### 2.1 手机盾

在手机银行中开通**手机盾**功能并设置密码。在国外进行交易时，需要使用手机盾进行安全验证。

同时，确保手机号已开通**国际漫游**功能，以便在国外能正常接收短信验证码。

### 2.2 线下刷卡设置

路径：手机银行 $\rightarrow$ 我的 $\rightarrow$ 我的账户 $\rightarrow$ 选中跨境通卡 $\rightarrow$ **限额设置**

需要开通以下两项功能：

| 功能 | 用途 |
|------|------|
| 小额/凭密码免密 | 线下刷卡时无需输入密码 |
| 磁条交易限额设置 | 支持磁条卡刷卡场景 |

### 2.3 线上支付设置

路径：手机银行 $\rightarrow$ 我的 $\rightarrow$ 我的账户 $\rightarrow$ 选中跨境通卡 $\rightarrow$ **支付设置**

需要开通以下功能：

| 功能 | 用途 |
|------|------|
| 电子支付-手机支付 | 手机端线上支付 |
| 电子支付-网上支付 | 网页端线上支付 |
| 跨行支付 | 跨行交易场景 |

> [安全] **安全提示**：以上功能建议**随用随开，用完后及时关闭**，保障用卡安全。

## 三、消费规则

### 3.1 余额要求

跨境通信记卡内余额需 **$\geq$ 支付金额的 102%**，否则支付会失败。

消费时，系统会按消费金额的 **102%** 冻结资金；账务清算结束后，多余的 2% 会自动解冻。

### 3.2 支持的货币

跨境通信记卡**不支持存入人民币**，仅支持以下 **19 种外币**：

| | | | |
|---|---|---|---|
| 美元 (USD) | 欧元 (EUR) | 日元 (JPY) | 港币 (HKD) |
| 英镑 (GBP) | 加拿大元 (CAD) | 澳大利亚元 (AUD) | 瑞士法郎 (CHF) |
| 新加坡元 (SGD) | 澳门元 (MOP) | 韩国元 (KRW) | 丹麦克朗 (DKK) |
| 挪威克朗 (NOK) | 新西兰元 (NZD) | 菲律宾比索 (PHP) | 俄罗斯卢布 (RUB) |
| 瑞典克朗 (SEK) | 泰国铢 (THB) | 南非兰特 (ZAR) | |

## 四、完整操作流程


### 文字版流程

1. **开卡**：办理一类银联借记卡 + 开通手机银行
2. **申办跨境通卡**：柜台开立跨境通信记卡（VISA / MasterCard）
3. **关联**：手机银行「账户管理」中关联跨境通卡
4. **安全设置**：开通手机盾，手机号开通国际漫游
5. **功能开通**：根据需要开通线下/线上支付功能
6. **购汇存款**：在手机银行中购买所需外币并存入跨境通卡
7. **境外使用**：刷卡消费或取现
8. **安全收尾**：使用完毕后及时关闭支付功能

## 五、常见问题

**Q：为什么支付失败了？**
A：请检查卡内余额是否 > 消费金额的 102%;是否已开通对应的支付功能;手机盾是否正常工作。

**Q：可以存入人民币吗？**
A：不可以。跨境通信记卡仅支持 19 种外币。需要先在手机银行中**购汇**（用人民币购买外币），再将外币存入该卡。

**Q：境外取现有什么限制？**
A：跨境通信记卡为**储蓄卡**，取现或消费前必须先存入相应币种。具体额度限制请咨询发卡银行。

**Q：回国后需要关闭哪些功能？**
A：建议关闭小额免密、磁条交易、电子支付等功能，降低盗刷风险。
